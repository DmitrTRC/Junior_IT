# Junior IT

Курс программирования для подростков 11–14 лет при ЮДО (юношеский добровольческий отряд) на территории Отряда территориальной обороны при 128 отделе полиции, Всеволожский район Ленобласти.

## Стек

Стартуем с **HTML + CSS** в браузере — нулевой порог входа, мгновенный визуальный результат, естественный мост к JavaScript.

Дальше: JavaScript → деплой в интернет → проекты.

## Структура

```
junior-it/
├── lessons/                # уроки по порядку
│   └── lesson-01-html-css/
│       ├── README.md       # о чём урок
│       ├── scenario.md     # сценарий с таймингом
│       ├── live-code.md    # пошаговый код для эфира
│       ├── glossary.md     # словарик терминов
│       ├── cheatsheet.html # шпаргалка для детей
│       ├── homework.html   # домашка-стартер
│       ├── slides.html     # слайды для шаринга экрана
│       ├── index-final.html # резервный финальный код
│       └── demos/          # эталонные демо
│           ├── gamer.html
│           ├── sport.html
│           └── art.html
├── docs/                   # общие материалы курса
│   ├── setup-guide.html    # как готовить устройство
│   └── soft-vs-hard-skills.html
├── students/               # работы учеников (gitignored)
└── README.md
```

## Формат занятий

- 75 минут, вебинар (MAX / Zoom / Meet)
- На занятии дети **смотрят и слушают** — параллельно не повторяют
- Делают сами дома по шпаргалке и домашке
- Раз в неделю
