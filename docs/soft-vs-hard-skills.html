Soft skills vs
Hard skills

Два слова, которые часто слышишь, но никто толком не объясняет. Сейчас объясню — за 3 минуты.

Когда говорят про «навыки» — обычно имеют в виду что-то одно из двух. И эти две штуки совершенно разные, хотя обе про тебя и твои умения.

Самое полезное — научиться их различать. Потому что качаются они тоже по-разному.

// hard skill

Что я умею ДЕЛАТЬ

Конкретное умение. Включил — сделал. Можно показать пальцем и проверить за 2 минуты.
Примеры
  • Решить квадратное уравнение
  • Забить штрафной с 11 метров
  • Сыграть «Кузнечика» на гитаре
  • Написать сайт на HTML
  • Сделать сальто назад
  • Разговаривать на английском
  • Собрать кубик Рубика за минуту
как проверить Дать задачу. Засечь время. Посмотреть на результат. Получилось / не получилось / насколько круто.
// soft skill

Какой я С СОБОЙ и С ЛЮДЬМИ

Привычки и качества характера. Их не покажешь за 2 минуты — они проявляются в долгую.
Примеры
  • Не сдаваться, когда не получается с первого раза
  • Спрашивать, когда непонятно (а не молчать в кулак)
  • Договариваться, когда поссорились
  • Слушать, а не только говорить
  • Держать слово — обещал → сделал
  • Помочь другу, не ожидая ничего взамен
  • Признать, когда был неправ
как проверить Тестом не проверишь. Видно только в реальной ситуации — когда трудно, когда устал, когда давит время.

А мы тут — про что качаем?

Наш курс — это HTML, CSS, скоро JavaScript. Это всё hard skills. Конкретные умения, которые можно показать: «вот, я сделал сайт». Хорошо, нужно.

// HTML // CSS // JavaScript // Git // деплой сайта

Но! Пока ты учишься кодить — ты параллельно качаешь кучу soft skills. И они вообще-то ценнее. Просто их не видно.

не сдаваться когда баг не находится читать чужой код задавать вопросы вместо угадывания разбивать большое на маленькое признавать ошибку
Программирование само по себе — это тренажёр. Не только для пальцев, но и для головы и характера. Это его главный плюс.

Когда у тебя 3 часа не работает один кусок кода, и ты не бросаешь, а ищешь — это soft. Когда наконец нашёл — это hard. Одно без другого не работает.

Главное

Hard и soft — не противники. Они работают парой. Hard без soft — «умный, но с ним не сработаешься». Soft без hard — «хороший человек, но ничего не умеет».

На каждом уроке ты качаешь и то, и другое. Замечай это. // you level up both
