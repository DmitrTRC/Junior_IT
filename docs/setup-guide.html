Подготовка устройства
к занятию

Короткий гид: 5–15 минут на настройку, и в четверг всё работает. Выберите свою платформу ниже.

📌 Несколько слов сначала

На занятии не нужно ничего повторять параллельно. Урок устроен так, что ребёнок смотрит и слушает — а делать сам будет дома, по шпаргалке и заданию, которые я пришлю в конце.

Поэтому минимальная задача — просто хорошо видеть и слышать. Если есть компьютер — отлично, можно сразу подготовиться и к домашке. Если только телефон — тоже нормально, шпаргалку откроет с компьютера попозже.

⬇ Выберите устройство, с которого ребёнок будет смотреть
📱 Режим: смотрим урок

1Конференц-приложение

За день до урока я пришлю ссылку. Чтобы по ней сразу зайти, на iPhone/iPad нужно поставить нужное приложение из App Store:

  • Zoom — если ссылка вида zoom.us
  • Google Meet — если meet.google.com
  • MAX — если будем созваниваться там (по умолчанию)

Какое именно — будет в моём сообщении. Поставьте заранее, чтобы не суетиться за 5 минут до старта.

ℹ️ MAX — это российский мессенджер. В App Store ищется по слову «MAX». Если у вас его пока нет — поставьте сразу, мы и дальше будем там общаться по группе.

2Настройки для качества

Звук

  • Лучше — наушники. Без них динамики айфона тихие, ребёнок устанет напрягаться.
  • Если наушники проводные — на новых iPhone нужен переходник на Lightning или USB-C.
  • AirPods и любые Bluetooth тоже отлично.

Экран

  • iPad предпочтительнее — больше экрана, удобнее видеть код.
  • Если iPhone — поверните горизонтально (ландшафт). Так картинка в Zoom больше.
  • Поставьте телефон на подставку или прислоните к чему-то — держать в руке 75 минут утомительно.

Чтоб не отвлекало

  • Включите «Не беспокоить»: Настройки → Фокус → Не беспокоить.
  • Закройте мессенджеры, игры, TikTok — на 75 минут.

3Что НЕ получится с iPhone/iPad

⚠ На iPad/iPhone полноценно кодить нельзя. Можно ставить разные приложения «для программирования», но они или платные, или не подходят для нашего урока. Это не страшно — ребёнок просто смотрит. Делать сам будет дома на компьютере.

Если компьютера дома вообще нет — напишите мне в личку, придумаем, как быть с домашкой (есть несколько вариантов, в том числе бесплатные онлайн-редакторы).

✓Проверь себя за 30 секунд

Откройте конференц-приложение (Zoom/Meet/MAX) и убедитесь:

  • Камера работает — видите себя
  • Микрофон работает — голос ловит
  • Звук слышно — хоть из наушников, хоть из динамика

За день до урока я пришлю ссылку. Чтобы по ней зайти, на Android нужно поставить нужное приложение из Google Play или RuStore:

  • Zoom — если ссылка вида zoom.us
  • Google Meet — если meet.google.com
  • MAX — если будем созваниваться там (по умолчанию)

Какое именно — будет в моём сообщении.

ℹ️ MAX — это российский мессенджер. Ищется в Google Play или RuStore по слову «MAX». Если у вас его пока нет — поставьте сразу, мы и дальше будем там общаться по группе. Если Google Play не открывается — пользуйтесь RuStore, там есть и MAX, и Zoom, и почти всё, что нужно. Ставится с rustore.ru.

2Настройки для качества

Звук

  • Желательно наушники — проводные с разъёмом или Bluetooth.
  • Если телефон Samsung/Xiaomi новый — у многих нет аудиоразъёма, нужен переходник Type-C.

Экран

  • Если есть планшет — он удобнее телефона.
  • На телефоне — поверните горизонтально.
  • Поставьте на подставку. Держать в руке 75 минут — тяжело.

Чтоб не отвлекало

  • Включите «Не беспокоить» — обычно в шторке уведомлений значок луны или Z.
  • Закройте всё лишнее.

3Что НЕ получится с Android

⚠ На телефоне Android полноценно кодить неудобно. Технически есть редакторы (типа Acode), но это не для начала. На занятии достаточно смотреть. Делать сам ребёнок будет дома на компьютере.

Если компьютера дома вообще нет — напишите мне, придумаем (есть бесплатные онлайн-редакторы, которые работают прямо в браузере).

💻 Режим: смотрим + готовимся делать

1Конференц-приложение

Поставьте заранее то, в чём будем созваниваться (по ссылке, которую я пришлю):

Zoom для Windows ↗ MAX для Windows ↗

На сайте max.ru — кнопка «Скачать» вверху. Google Meet ставить не нужно — он работает прямо в браузере.

2Браузер

Нужен современный браузер: Chrome, Edge или Яндекс.Браузер. На Windows 10/11 Edge уже стоит — этого достаточно. Если хотите Chrome — скачайте с google.com/chrome.

⚠ Internet Explorer не подойдёт. На нём современные сайты выглядят сломанными. Он есть только на старых Windows 7/8.

3VS Code — программа для кода

Это «умный блокнот» программистов. Бесплатный, легальный, ставится за 3 минуты:

Скачать VS Code ↗

На странице — большая кнопка Windows. Жмите, дальше:

  1. Откройте скачанный файл (обычно VSCodeUserSetup-x64-...exe).
  2. Принимаете лицензию → Далее.
  3. Папка установки — по умолчанию, Далее.
  4. Важно: поставьте галочки «Добавить действие "Открыть в VS Code"» и «Добавить в PATH». Так удобнее.
  5. Далее → Установить. 2 минуты ждём.
  6. В конце — галочка «Запустить» → Готово.

4Live Server — авто-обновление страницы

Маленькое расширение, которое сильно облегчает жизнь: после сохранения файла страница в браузере обновляется сама.

  1. В VS Code слева — иконка с квадратиками (или Ctrl + Shift + X).
  2. В поиске сверху наберите: Live Server
  3. Первый в списке — от Ritwick Dey (миллионы установок, синяя ракета на иконке).
  4. Нажмите кнопку Install.

5Создаём рабочую папку

  1. На рабочем столе — правый клик → Создать → Папку. Назовите её site (английскими буквами, без пробелов).
  2. Откройте VS Code → меню File → Open Folder → выберите папку site → Выбор папки.
  3. В левой панели VS Code увидите название папки.
  4. Наведите курсор на имя папки — появятся иконки. Нажмите «Новый файл» (значок листа с плюсом).
  5. Назовите файл: index.html и нажмите Enter.
💡Зачем именно index.html Это стандартное имя главной страницы любого сайта. Браузер открывает его автоматически.

✓Проверь себя за 60 секунд

Скопируйте этот код в свой index.html:

<style>
  body { background: #1a1a2e; color: #00e676;
         padding: 60px; text-align: center;
         font-family: sans-serif; }
  h1 { font-size: 50px; }
</style>
<h1>✓ Я готов к уроку!</h1>

Сохраните: Ctrl + S. Внизу справа в VS Code нажмите «Go Live».

Откроется браузер. Если видите большое зелёное «✓ Я готов к уроку!» на тёмном фоне — всё работает.

Частые вопросы

У нас только телефон, компьютера дома нет
Не страшно для занятия — смотреть с телефона норм. Для домашки есть варианты: можно через бесплатный онлайн-редактор прямо в браузере. Напишите мне в личку — подскажу, что именно подойдёт под ваш случай.
Компьютер очень старый, потянет?
VS Code работает даже на слабых машинах. Если ноут 5–10 лет назад выпуска и у него хотя бы 4 ГБ памяти — справится. Если совсем старый и тормозит — пишите, попробуем варианты полегче.
Можно ребёнку одному, без меня?
Лучше первые 10 минут — рядом. Помочь подключиться, проверить звук. Дальше — спокойно занимайтесь своими делами, я держу группу 75 минут самостоятельно.
Нужно ли что-то платить или подписывать?
Всё бесплатно и легально. VS Code, Live Server, Chrome, Zoom — все программы официальные, без подписок и регистраций. Никакого «прокачать аккаунт» — для нашего урока ничего такого не нужно.
Что делать, если на занятии что-то не запустится?
Не паникуем. На самом уроке параллельно повторять не нужно — ребёнок просто смотрит. Если что-то не работает на устройстве, после урока разберёмся индивидуально, не теряя занятие.
Камеру обязательно включать?
Желательно — мне важно видеть лица, чтобы понимать, кто залип. Но если ребёнок стесняется или утром не успел причесаться — можно без камеры, не давим.
Сколько по времени всё это устанавливать?
На телефон — 5 минут. На Windows/Mac, если делать с нуля (Zoom + VS Code + Live Server + проверочный файл) — 15–20 минут спокойно. Лучше не откладывать на четверг до старта.

Что-то не получается, какой-то шаг непонятен или странная ошибка —

просто напишите мне в личку, разберёмся.
