# Правила общения

Junior_IT — курс для подростков, и пространство вокруг него (issues, обсуждения,
чаты занятий) должно быть таким, куда не страшно прийти новичку любого возраста.
Правила короткие и человеческие.

## По-доброму — это как

- **Уважаем друг друга.** Любой вопрос — нормальный. «Глупых вопросов» тут нет.
- **Критикуем код и идеи, а не людей.** «Вот здесь ошибка» — да. «Ты бестолковый» — нет.
- **Помогаем, а не высмеиваем.** Кто-то залип на простом — подскажи, не подкалывай.
- **Бережём чужое личное.** Не выкладываем чужие имена, фото, контакты, переписку
  без согласия. Особенно — данные детей.

## Так нельзя

- Травля, оскорбления, угрозы, переход на личности.
- Дискриминация по любому признаку.
- Материалы 18+, жестокость, разжигание вражды.
- Публикация чужих личных данных.
- Спам и реклама не по теме.

## Что может сделать ведущий

Автор курса вправе убрать неуместный комментарий, закрыть обсуждение или
ограничить участника, который нарушает эти правила. Цель одна — чтобы место
оставалось дружелюбным к детям, которые только начинают.
