## 💡 Решения, которые приняли по ходу

| Решение | Причина |
|---|---|
| Запись урока, не только live | Страховка от технических проблем + актив для следующих наборов |
| Имя репо `Junior_IT` (PascalCase) | По решению — соответствует имени курса |
| Скейтер в demos переименован Ника → Саня | По запросу |
| Свет Elgato: 5800K + 35% яркости | Компенсирует красноту лица от патрулей |
| Дресс-код: нейтральная тёмная одежда без надписей | Записи на YouTube будут жить годами — никаких политических сигналов |
| Camera lock-all (focus/WB/ISO/exposure) | Картинка не должна гулять во время записи |
| Шаринг правого монитора целиком, не Application | macOS Spaces ломают Application-шаринг при переключении десктопов |
