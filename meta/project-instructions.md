## 🎯 О проекте

**Junior_IT** — IT-направление при ЮДО на территории Отряда территориальной обороны при 128 отделе полиции, Всеволожский район Ленобласти.

Ведёт Димас. По стеку — senior-разработчик. Junior_IT — одно из направлений работы с подростками, наряду с лесом/ориентированием, первой помощью, ЧС.

**Аудитория:** дети 11–14 лет.

**География:** посёлки Берёзовка, Мечта, Синицино, Хязельки (Лукоморье), Воейково, Каниста, Кирполье, Покосы, Родничковое.

**Социальный микс:** от деревенских до мажоров из Лукоморья. Это важно для тона коммуникации с родителями.

**Формат:** вебинар 75 минут через MAX мессенджер по умолчанию, иногда Zoom/Meet. Запись урока — в Ecamm Live на Mac, для тех кто не подключился вживую и для архива.

## ⚙️ Методологические принципы

- Стартуем с **HTML+CSS** (не Python, не C). Причины: нулевой барьер входа, мгновенный визуальный результат, девочки не отваливаются, естественный мост к JS.
- На вебинаре дети **только смотрят и слушают**, НЕ повторяют параллельно. Делают дома по шпаргалке и стартеру.
- Никакой теории дольше 2 минут подряд.
- Каждые 3–5 минут — пинг в чат, вопрос, реакция.
- CSS — кульминация первого урока, не приложение к HTML.
- НЕ используются в первых 3–4 уроках: «семантика», «DOCTYPE», «валидация», «парсинг», «DOM».
- На каждый урок — резервный финальный файл (если живой кодинг сломается).
- Эталонных демок несколько (геймер/спорт/арт) — каждый ребёнок узнаёт себя.
- 75 минут — потолок (после внимание мертво).

## ⛔ Запреты

### В материалах для детей
- **НЕ использовать** символику СССР, ФСБ, силовых ведомств, советских органов, любые политически нагруженные принты.
- **НЕ говорить** слово «дружина» (взрослые ассоциируют с СССР, дети не понимают).
- **НЕ упоминать** «контроль», «проверки», «инспекции» в материалах для родителей.
- **НЕ предлагать** «кружковую методику как в школе».

### В архитектурных решениях
- **НЕ давать слишком много** за раз — Димас сам остановит и попросит развёрнуто.
- **НЕ лезть** с альтернативами выбранному стеку без явного запроса.
- **НЕ переписывать** окружающий контекст при правках — только хирургические.

- Кастомные SVG-баннеры вместо генераторов вроде capsule-render
- В Markdown — эмодзи-заголовки, бейджи через img.shields.io

## 🎥 Сетап для записи уроков

### Конфигурация (mid-2026)
- **Камера:** Sony ZV-E10 + 16-50 PZ → Elgato Cam Link 4K → Mac Studio
- **Аудио:** Shure SM7B → Союз Лончер → UAD Apollo Twin X DUO → UAD Console → Ecamm
- **Свет:** 2× Elgato Key Light, ~5800K, ~35% яркости, 45° по бокам
- **Запись:** Ecamm Live, Record Only mode, isolated tracks, MOV
- **Папка записей:** `~/Movies/Junior_IT_Recordings/`
- **Сцены Ecamm:** INTRO / CODE (правый монитор, шаринг дисплея) / BROWSER / DEMO

### Камера: ручные настройки
- Focus: **Manual**, залочен по пикингу
- WB: **Manual ~5500K** (под холодные Elgato)
- ISO: **фиксированный** (не Auto)
- Shutter: **1/50** (для 25p)
- Aperture: **F3.5**
- HDMI Info Display: **Off**

### Дресс-код
- Нейтральная тёмная одежда **без надписей**
- **Запрещено** в кадре: символика СССР, ФСБ, силовых ведомств, любая политически нагруженная одежда
- Это касается записей для детей — в быту/на патрулях Димас носит что хочет
