циклы · while / for · урок 2.7

01Цикл: повтори несколько раз

Цикл — команда «повтори то, что в фигурных скобках, нужное число раз». Компьютер не устаёт: что 5 раз, что 5000. Считает повторы коробочка-счётчик i.

// руками — скучно и тупоconsole.log("Привет 1")console.log("Привет 2")// …а если тысячу раз? нужен цикл

02while — повторяй, пока…

В круглых скобках — условие (как у if, только повторяется). В фигурных — что повторять. Счётчик i ведём руками: завести, проверить, увеличить.

let i = 1while (i <= 5) {  console.log("Привет " + i)  i = i + 1}// напечатает Привет 1 … Привет 5
let i = 1
Завести счётчик — с какого числа считаем.
i = i + 1
Увеличить счётчик. Без этой строки цикл не закончится!

⚠️Главная ловушка: бесконечный цикл

Если забыть увеличить счётчик, i навсегда останется 1, условие всегда true — и цикл крутится без конца. Вкладка виснет.

// НЕ запускай — для понимания!let i = 1while (i <= 5) {  console.log("застрял на " + i)  // забыли i = i + 1 → беда}
🔑
Внутри цикла всегда меняй счётчик. Завис? Закрой вкладку. У for шаг встроен в скобки — поэтому забыть его сложнее.

03Цикл умеет считать

Внутри можно не только печатать, но и копить. Две коробочки: i считает повторы, sum копит результат.

let i = 1let sum = 0while (i <= 5) {  sum = sum + i  i = i + 1}console.log("Сумма: " + sum)Сумма: 15

04for — та же тройка в одну строку

В круглых скобках через ; сразу три части: старт; условие; шаг. То же, что в while, только собрано в строку — и забыть про шаг уже сложно.

for (let i = 1; i <= 5; i++) {  console.log("Привет " + i)}// тот же результат, что у while
i++
Шаг: короткая запись i = i + 1 — «увеличь на один».
i--
Уменьши на один — для обратного отсчёта.

05Спроси и повтори

Соединяем с прошлыми уроками: спросить число через prompt + Number и прогнать цикл. Программа сама печатает таблицу умножения.

let n = Number(prompt("Какое число умножаем?"))for (let i = 1; i <= 10; i++) {  console.log(n + " × " + i + " = " + (n * i))}// ввёл 7 → вся таблица на семь
🧮
Скобки вокруг (n * i) нужны: сначала посчитай, потом приклеивай к тексту. Без них браузер склеит по буквам.

06Обратный отсчёт и узор (для сильных)

Счётчик можно гнать вниз (i--) или копить строку.

// обратный отсчёт 5 → 1for (let i = 5; i >= 1; i--) { console.log(i) }// узор: лесенка из звёздlet uzor = ""for (let i = 1; i <= 5; i++) { uzor = uzor + "⭐"; console.log(uzor) }

07Топ-5 ошибок новичка

1. Забыл шаг счётчика. Нет i = i + 1 / i++ → бесконечный цикл, вкладка виснет.
2. Условие через =. Нужно i <= 5 (сравнить), а не i = 5 (положить).
3. Запятые в for. Между тремя частями — ; (точка с запятой), не запятая.
4. Кавычки-«ёлочки». Только английские " или ', не « и не ».
5. Русская раскладка. while, for — английскими буквами.
🛠
Если вкладка зависла — почти всегда забыт шаг счётчика. Закрой вкладку и добавь i++.

08Что дальше

Теперь у тебя весь чистый язык: программа хранит (переменные), выбирает (if) и повторяет (циклы). На следующем уроке вернём кнопки — и всё это оживёт на клике: реакции и мини-игры.
