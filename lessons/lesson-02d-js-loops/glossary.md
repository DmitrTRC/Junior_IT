# 📖 Словарик урока 2.7 — «Программа повторяет: циклы»

> Объясняем простыми словами, без «итерация/тело цикла/инкремент/массив». Эти слова — позже. Сегодня только: повторять и считать повторы.

| Слово | Простыми словами | Пример |
|---|---|---|
| **цикл** | Команда «повтори это несколько раз». Браузер сам прогоняет одни и те же строки много раз. | `while`, `for` |
| **`while (...)`** | «Пока…». Повторяй то, что в фигурных скобках, **пока** условие в круглых скобках `true`. | `while (i <= 5) { ... }` |
| **счётчик `i`** | Коробочка, которая считает, какой это повтор: 1, 2, 3… Заводим до цикла, увеличиваем внутри. | `let i = 1` |
| **`i = i + 1`** | «Увеличь счётчик на один». Без этой строки цикл не закончится — повторяй не забывай! | `i = i + 1` |
| **`for (...)`** | Краткая запись `while`. В одних скобках сразу три части: **старт**; **условие**; **шаг**. | `for (let i = 1; i <= 5; i++)` |
| **`i++`** | Короткая запись `i = i + 1` — «увеличь на один». Живёт в `for`. | `for (... ; i++)` |
| **`i--`** | «Уменьши на один». Нужна для обратного отсчёта. | `for (let i = 5; i >= 1; i--)` |
| **`{ }` фигурные скобки** | «Вот это и повторяй» — что внутри, то и крутится каждый раз. | `{ console.log(i) }` |
| **бесконечный цикл** | Цикл, который не заканчивается, потому что условие всегда `true` (забыли увеличить счётчик). Вкладка виснет. | `while (i <= 5) { }` без `i = i + 1` |

---

## Главная мысль урока одной строкой

`if` делает что-то **один раз**, а цикл **повторяет** — пока счётчик не дойдёт до конца. Компьютер не устаёт: что повтори 5 раз, что 5000.

## Главная ловушка, из-за которой «виснет»

**Забыл увеличить счётчик.** Если внутри `while` нет строки `i = i + 1`, то `i` навсегда останется `1`, условие `i <= 5` всегда `true`, и цикл крутится без конца — вкладка зависает. Лечение: внутри цикла обязательно меняем счётчик. В `for` шаг (`i++`) встроен в скобки — поэтому новичку с `for` безопаснее.
