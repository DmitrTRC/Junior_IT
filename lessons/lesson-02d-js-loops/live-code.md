# 💻 LIVE-CODE: «Программа повторяет — циклы» (урок 2.7)

**Стартовая точка:** НИКАКИХ файлов. Открываешь браузер → консоль (F12 → Console). Всё пишем прямо там.
**Зачем этот урок:** на прошлых уроках каждое действие случалось **один раз** — посчитал, сравнил, решил. Сегодня учим программу **повторять**: отсчитать, напечатать таблицу, нарисовать ряд за рядом. Всё ещё чистый язык в консоли, без страницы и кнопок. Кнопки вернём на следующем уроке.
**Принцип:** одна строка = один ответ браузера. Между шагами — пинг в чат. Теории не дольше 2 минут подряд.

## ШАГ 0 · Напоминаем, где мы

🎬 Открываешь вкладку → `F12` → вкладка **Console**. Курсор после `>`. Печатаешь `2 + 2` → `4`.

🎤 «За три урока программа научилась считать, хранить в коробочках и выбирать дорогу через `if`. Но всё это она делает по разу. А компьютер силён тем, что не устаёт **повторять**. Сегодня про это. И снова хватит одной консоли.»

💬 **В чат:** «Что в играх повторяется снова и снова? Кидайте — волны врагов, тики таймера, шаги?»

> ⚠️ Снова покажи, КУДА печатать (вкладка Console). Дома путают с Elements — главный «не работает».

## ШАГ 1 · Повторять руками — скучно

🎤 «Допустим, надо поздороваться пять раз. В лоб — вот так.»

```js
console.log("Привет 1")
console.log("Привет 2")
console.log("Привет 3")
console.log("Привет 4")
console.log("Привет 5")
```

🎬 Печатается пять строк. Ты устало вздыхаешь.

🎤 «А если тысячу раз? Руками — никогда. Должна быть команда "повтори это N раз". Она есть — называется цикл.»

💬 **В чат:** «Сколько строк надоело бы печатать вручную? 10? 100?»

---

## ШАГ 2 · `while` — повторяй, пока…

🎤 «Главное слово: `while` — "пока". Заведём коробочку-счётчик и скажем: пока счётчик не дошёл до 5 — повторяй.»

```js
let i = 1
while (i <= 5) {
  console.log("Привет " + i)
  i = i + 1
}
```

🎬 Печатается «Привет 1 … Привет 5». Сами, одной командой.

🎤 «Разберём по кусочкам, все знакомые:
— `let i = 1` — завели счётчик (коробочка из урока 2.5);
— `while (i <= 5)` — условие, как в `if`, только повторяется (урок 2.6);
— `i = i + 1` — увеличили счётчик, иначе застрянем.»

💬 **В чат:** «🔁 если видно, как одна команда сделала пять строк.»

> ⚠️ Круглые скобки — условие, фигурные — что повторять. Ровно как у `if`, только крутится.

---

## ШАГ 3 · Бесконечный цикл (главная ловушка)

🎤 «А теперь — самая коварная ошибка урока. Что будет, если забыть увеличить счётчик?»

```js
// НЕ запускай это насовсем — для показа!
let i = 1
while (i <= 5) {
  console.log("застрял на " + i)
  // забыли i = i + 1
}
```

🎬 Браузер сыплет «застрял на 1» без конца, вкладка виснет.

🎤 «`i` навсегда остался единицей, `i <= 5` всегда "да" — и цикл крутится вечно. Это бесконечный цикл. Лечение: закрыть вкладку. А чтобы не повторилось — **внутри цикла обязательно меняем счётчик**.»

💬 **В чат:** «😱 кто понял, почему он не остановился?»

> ⚠️ Это главная грабля урока — аналог `=`/`===` с прошлого занятия. Покажи живьём (быстро закрой вкладку), чтобы дети не пугались дома.

---

## ШАГ 4 · Цикл умеет считать

🎤 «Внутри цикла можно не только печатать, но и копить. Сложим числа от 1 до 5.»

```js
let i = 1
let sum = 0
while (i <= 5) {
  sum = sum + i
  i = i + 1
}
console.log("Сумма: " + sum)
```

🎬 Ответ: `Сумма: 15`.

🎤 «Две коробочки: `i` считает повторы, `sum` копит результат. Цикл прибавил 1, потом 2, 3, 4, 5 — и сам сложил. Руками так считать не пришлось.»

💬 **В чат:** «А сумму от 1 до 100 цикл осилит? (да, мгновенно)»

---

## ШАГ 5 · `for` — та же тройка в одну строку

🎤 «В `while` про счётчик три заботы: завести, проверить, увеличить. Их можно собрать в одну строку — это `for`.»

```js
for (let i = 1; i <= 5; i++) {
  console.log("Привет " + i)
}
```

🎬 Тот же результат: «Привет 1 … Привет 5».

🎤 «Смотрите, в круглых скобках через `;` три части — те же самые:
— `let i = 1` — **старт**;
— `i <= 5` — **условие**;
— `i++` — **шаг** (это короткая запись `i = i + 1`).
Ничего нового — просто счётчик собрали в одну строку. И забыть про шаг уже сложнее: он на виду.»

💬 **В чат:** «🔥 если `for` понятен как сокращённый `while`.»

> ⚠️ Точка с запятой `;` между тремя частями — не запятая. Покажи все три части пальцем по очереди.

---

## ШАГ 6 · Таблица умножения (кульминация)

🎤 «Соберём что-то настоящее. Спросим число — и программа сама напечатает на него таблицу умножения.»

```js
let n = Number(prompt("Какое число умножаем?"))
for (let i = 1; i <= 10; i++) {
  console.log(n + " × " + i + " = " + (n * i))
}
```

🎬 Вводишь `7` → печатается вся таблица на семь, от `7 × 1 = 7` до `7 × 10 = 70`.

🎤 «`Number` снова обязателен — `prompt` отдаёт текст. А дальше цикл прогнал `i` от 1 до 10 и на каждом шаге посчитал `n * i`. Десять строк — одной командой.»

💬 **В чат:** «На какое число запустить таблицу? Кидайте — наберу вживую.»

> ⚠️ Скобки вокруг `n * i` нужны: без них браузер сначала склеит текст. `(n * i)` — посчитай, потом приклей.

## ШАГ 7 · Обратный отсчёт и узор (для вариативности / сильных)

🎤 «Счётчик можно гнать и вниз — для обратного отсчёта. Меняем `i++` на `i--`.»

```js
for (let i = 5; i >= 1; i--) {
  console.log(i)
}
console.log("Поехали! 🚀")
```

🎬 Печатается `5, 4, 3, 2, 1, Поехали!`.

🎤 «А ещё цикл умеет рисовать. Будем копить строку из звёздочек.»

```js
let uzor = ""
for (let i = 1; i <= 5; i++) {
  uzor = uzor + "⭐"
  console.log(uzor)
}
```

🎬 Растёт лесенка: `⭐`, `⭐⭐`, `⭐⭐⭐`, `⭐⭐⭐⭐`, `⭐⭐⭐⭐⭐`.

> 💡 Скучающему («я знаю циклы») — задача: «напечатай только чётные числа от 2 до 20» (`i = i + 2` или `i += 2`).

## ИТОГ на экране (что прозвучало)

- Цикл — команда «повтори несколько раз», компьютер не устаёт
- `while (условие) { ... }` — повторяй, пока условие `true`; счётчик ведём руками
- Счётчик: завести `let i = 1` → проверить `i <= 5` → увеличить `i = i + 1`
- `for (let i = 1; i <= 5; i++)` — та же тройка (старт; условие; шаг) в одну строку
- `i++` увеличить, `i--` уменьшить (обратный отсчёт)
- Ловушка: забыл менять счётчик → **бесконечный цикл**, вкладка виснет

## 🆘 Если живой кодинг встал

1. **Вкладка зависла** → бесконечный цикл: внутри нет `i = i + 1` (или `i++`). Закрой вкладку, добавь шаг.
2. **Цикл не повторяется / выполнился раз** → проверь условие: `i <= 5`, а не `i = 5`. И счётчик стартует с правильного числа.
3. **«Красная ошибка»** → в `for` между частями `;` (точка с запятой), не запятая. Ещё частое — кавычки-ёлочки и русская раскладка.
4. **Совсем сломалось** → открой `index-final.html`, дальше веди по нему: там готовые строки с ответами.
