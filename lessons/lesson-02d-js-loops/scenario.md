### 🟡 [0:16 – 0:28] `while` — повторяй, пока… (кульминация №1)

🎤 «Главное слово: `while` — "пока". Заводим счётчик и крутим, пока он не дошёл до пяти.»

```js
let i = 1
while (i <= 5) {
  console.log("Привет " + i)
  i = i + 1
}
```

🎬 Пять строк — одной командой.

🎤 «Всё знакомое: `let i = 1` — коробочка-счётчик (урок 2.5); `while (i <= 5)` — условие, как у `if`, только повторяется (2.6); `i = i + 1` — увеличили, иначе застрянем.»

💬 **В чат:** «🔁 если видно, как одна команда сделала пять строк.»

> ⚠️ Покажи три части счётчика пальцем: завести · проверить · увеличить. Тут «щёлкает» весь урок.

---

### 🟡 [0:28 – 0:36] Бесконечный цикл (главная ловушка)

🎤 «А теперь самая коварная ошибка. Что если забыть увеличить счётчик?»

🎬 Запускаешь без `i = i + 1` — консоль сыплет без конца, вкладка виснет. **Быстро закрываешь вкладку.**

🎤 «`i` навсегда остался единицей, условие всегда "да" — крутится вечно. Это бесконечный цикл. Лечение — закрыть вкладку. Чтобы не повторилось: внутри цикла всегда меняем счётчик.»

💬 **В чат:** «😱 кто понял, почему он не остановился?»

> ⚠️ Это грабля №1 урока (как `=`/`===` в прошлый раз). Покажи живьём и спокойно — чтобы дома не пугались.

### 🟡 [0:36 – 0:44] Цикл умеет считать

🎤 «Внутри можно не только печатать, но и копить. Сложим числа от 1 до 5.»

```js
let i = 1
let sum = 0
while (i <= 5) {
  sum = sum + i
  i = i + 1
}
console.log("Сумма: " + sum)
```

🎬 Ответ: `Сумма: 15`.

🎤 «Две коробочки: `i` считает повторы, `sum` копит. Цикл прибавил 1, 2, 3, 4, 5 — и сам сложил.»

💬 **В чат:** «А сумму от 1 до 100 цикл осилит? (да, мгновенно)»

### 🟡 [0:44 – 0:52] `for` — та же тройка в одну строку (кульминация №2)

🎤 «Заводим, проверяем, увеличиваем — три заботы. Соберём их в одну строку: это `for`.»

```js
for (let i = 1; i <= 5; i++) {
  console.log("Привет " + i)
}
```

🎤 «Через `;` три части — те же: `let i = 1` старт, `i <= 5` условие, `i++` шаг (короткая запись `i = i + 1`). Ничего нового — счётчик собрали в строку, и забыть про шаг уже сложно.»

💬 **В чат:** «🔥 если `for` понятен как сокращённый `while`.»

> ⚠️ Между частями `;` — точка с запятой, не запятая. Покажи все три пальцем.

### 🟡 [0:52 – 0:58] Таблица умножения (главное за урок)

🎤 «Соберём настоящее. Спросим число — программа сама напечатает таблицу.»

```js
let n = Number(prompt("Какое число умножаем?"))
for (let i = 1; i <= 10; i++) {
  console.log(n + " × " + i + " = " + (n * i))
}
```

🎬 Вводишь `7` → вся таблица на семь, от `7 × 1 = 7` до `7 × 10 = 70`.

🎤 «`Number` обязателен — `prompt` отдаёт текст. Цикл прогнал `i` от 1 до 10, на каждом шаге посчитал. Десять строк — одной командой.»

💬 **В чат:** «На какое число запустить таблицу?»

### 🟡 [0:58 – 1:05] Итог + мостик вперёд

🎤 «Что сегодня:»

🎤 «Вот и весь чистый язык: программа хранит, выбирает и повторяет. На следующем уроке вернём кнопки — и всё это оживёт на клике: реакции, мини-игры.»
