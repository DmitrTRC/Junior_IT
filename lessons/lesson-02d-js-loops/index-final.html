1Повторять руками — скучно

console.log("Привет 1")console.log("Привет 2")// …а если тысячу раз? нужен цикл

2while — повторяй, пока (кульминация №1)

let i = 1while (i <= 5) {  console.log("Привет " + i)  i = i + 1}// Привет 1 … Привет 5
Счётчик: let i = 1 завести · i <= 5 проверить · i = i + 1 увеличить.

3Бесконечный цикл (главная ловушка)

// НЕ запускать насовсем — для показа!let i = 1while (i <= 5) {  console.log("застрял на " + i)  // забыли i = i + 1}
Счётчик не растёт → условие всегда true → крутится вечно. Лечение: закрыть вкладку.

4Цикл умеет считать

let i = 1let sum = 0while (i <= 5) {  sum = sum + i  i = i + 1}console.log("Сумма: " + sum)Сумма: 15

5for — та же тройка в строку (кульминация №2)

for (let i = 1; i <= 5; i++) {  console.log("Привет " + i)}// тот же результат, что у while
Через ; три части: let i = 1 старт · i <= 5 условие · i++ шаг (= i = i + 1).

6Таблица умножения (главное за урок)

let n = Number(prompt("Какое число умножаем?"))for (let i = 1; i <= 10; i++) {  console.log(n + " × " + i + " = " + (n * i))}
Ввёл 7 → от 7 × 1 = 7 до 7 × 10 = 70. Скобки (n * i) — посчитать, потом приклеить.

7Бонус: обратный отсчёт и узор

// вниз — i--for (let i = 5; i >= 1; i--) { console.log(i) }console.log("Поехали! 🚀")// узор: копим строкуlet uzor = ""for (let i = 1; i <= 5; i++) { uzor = uzor + "⭐"; console.log(uzor) }
