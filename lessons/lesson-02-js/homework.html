Оживи свой сайт

Берём страничку с прошлого урока и добавляем кнопки, которые реально работают. Шпаргалка рядом — всё получится.

1Открой свою страничку

Открой в VS Code свой index.html с прошлого урока. Будем добавлять к нему, не начинаем заново.

Не доделал прошлую домашку? Не страшно — возьми стартер ниже и работай с ним.

2Добавь счётчик

Вставь это в свою страничку — в любое место, где хочешь видеть счётчик:

<h2 id="score">0</h2> <button onclick="add()">+1</button>

А это — в самый низ, перед закрывающим </body> (или просто в самом конце файла):

<script> let score = 0; function add() { score = score + 1; document.getElementById('score').textContent = score; } </script>

Сохрани (Ctrl+S), обнови браузер (F5), жми кнопку — число должно расти. 🎉

3Добавь кнопку, которая меняет фон

Две кнопки — тёмная и светлая тема. Подставь свои любимые цвета:

<button onclick="document.body.style.background='#0f3460'">Тёмная</button> <button onclick="document.body.style.background='#faf7f2'">Светлая</button>

4Сделай это СВОИМ

  1. Поменяй надпись на кнопке счётчика (например «Я молодец +1»)
  2. Добавь вторую кнопку-всплывашку: onclick="alert('твой текст')"
  3. Подставь свои цвета в кнопки темы
  4. Придумай, что считать именно у тебя: голы, уровни, рисунки, лайки

✦ Идеи для прокачки

  • Кнопка «−1»: сделай функцию sub(), где score = score - 1
  • Кнопка «Сброс»: score = 0 и обнови текст
  • Меняй по клику не фон, а цвет заголовка: .style.color
  • Поменяй текст заголовка по клику через .textContent

5Покажи и пришли

  1. Покажи родителям — пусть нажмут на твою кнопку
  2. Сделай скриншот (Print Screen на Windows, Cmd+Shift+4 на Mac)
  3. Пришли скриншот мне в чат — посмотрю, дам совет
  4. На следующий урок принеси файл — будем добавлять движение

✓ Проверь себя

Если кнопка «+1» увеличивает число на странице, а кнопка темы меняет фон — задача выполнена. Красава.

Не работает?

Это нормально, у всех так бывает. Проверь по порядку:

→ <script> в самом низу страницы?
→ Имя "score" совпадает в id и в getElementById?
→ Кавычки английские, не «ёлочки»?
→ Зовёшь add() со скобками?
→ Сохранил (Ctrl+S) и обновил (F5)?

Нажми F12 → Console — красная строка подскажет, где ошибка. Совсем застрял — пиши мне.
