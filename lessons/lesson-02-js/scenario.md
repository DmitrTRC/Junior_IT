### 🟡 [0:10 – 0:14] Минимум теории (3 идеи, не больше 2 минут подряд)

🎤 «Три языка — три работы. **HTML** строит (теги). **CSS** красит (стили). А сегодняшний — **JavaScript** — заставляет ДЕЙСТВОВАТЬ.»

**Идея 1.** «JS пишется внутри тега `<script>`. Помните `<style>` для красоты? Тут так же, только для действий.»

**Идея 2.** «Действие вешаем на кнопку через слово `onclick` — буквально "при клике".»

**Идея 3.** «Чтобы что-то посчитать, заводим переменную — коробочку со значением. `let счёт = 0`.»

### 🟡 [0:14 – 0:25] Первая живая кнопка

🎬 VS Code + браузер рядом. В страничку урока 1 добавляешь кнопку и скрипт.

🎤 «`button` — кнопка. `onclick` — что сделать при клике. `alert` — показать сообщение. Три слова — и кнопка живая.»

### 🟡 [0:25 – 0:45] Счётчик (кульминация урока)

🎤 «Теперь самое крутое — счётчик. Считаем, сколько раз нажали.»

```html
<h2 id="score">0</h2>
<button onclick="add()">+1</button>

<script>
  let score = 0;
  function add() {
    score = score + 1;
    document.getElementById('score').textContent = score;
  }
</script>
```

🎬 Кликаешь — число растёт. **Момент "вау".**

🎤 «`id` — это имя элемента, чтобы JS его нашёл. `getElementById` — "найди по имени". `textContent` — "поменяй текст внутри". Каждый клик прибавляет 1.»

💬 **В чат:** «У кого в голове щёлкнуло, как это работает? 🔥»

> ⚠️ Самая частая ошибка дома — забыть `id` или написать имя по-разному в двух местах. В шпаргалке подсветить.

### 🟡 [0:45 – 0:58] Меняем стиль по клику + кнопка-сюрприз

```html

<button onclick="document.body.style.background='#faf7f2'">Светлая тема</button>
```

🎬 Клик — фон меняется.

🎤 «JS может трогать ЛЮБОЙ стиль, как CSS — только по нажатию. Дома попробуете свои цвета.»

💬 **В чат:** «Какие два цвета поставили бы на свои кнопки?»

### 🟡 [0:58 – 1:05] Сравниваем с демками, подводим итог

🎤 «Что сегодня узнали:»
1. JS живёт в `<script>` и отвечает за действия.

4. `getElementById` + `textContent` — найти элемент и поменять.
5. По клику можно менять и текст, и стиль.
