# Урок 2 — Оживляем сайт

> ⚠️ **ЧЕРНОВИК** (заготовка автозадачи от 6 июня 2026). Проверь, поправь под группу, потом убери эту плашку.

**Тема:** Кнопки, счётчики и всплывашки на JavaScript
**Длительность:** 75 минут
**Возраст:** 11–14 лет
**Уровень:** продолжение урока 1 (HTML + CSS уже знают)

## Что ребёнок умеет к концу урока

- Понимает, что JS — это язык «действий»: HTML строит, CSS красит, JS оживляет
- Знает, что код пишется внутри тега `<script>` (как `<style>`, только для действий)
- Умеет повесить действие на кнопку через `onclick`
- Понимает, что такое переменная (коробочка со значением) на примере счётчика
- Умеет менять текст и стиль элемента по клику (`textContent`, смена цвета)
- Может сделать дома: кнопку-счётчик и кнопку-«всплывашку» по шпаргалке
