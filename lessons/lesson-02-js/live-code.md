# 💻 LIVE-CODE: «Оживляем сайт» (урок 2)

> ⚠️ **ЧЕРНОВИК** (заготовка автозадачи от 6 июня 2026). Прогони руками от начала до конца — каждый шаг должен реально запускаться. Потом убери эту плашку.

**Стартовая точка:** НЕ пустой файл. Берём `index.html` ребёнка (или `index-final.html` урока 1) и оживляем ЕГО.
**Резерв:** если живой кодинг сломался — открываешь `index-final.html` этого урока, он уже рабочий.
**Принцип:** один шаг = одно «вау». Между шагами — пинг в чат. Теории не дольше 2 минут подряд.

---

## ШАГ 0 · Что на экране перед стартом

VS Code слева, браузер справа. Открыта страничка с урока 1 — заголовок, пара блоков, стили. Просто лежит, ничего не делает.

🎤 «Вот наша страничка с прошлого раза. Красивая, но мёртвая. Сейчас оживим — добавим первый язык действий.»

---

## ШАГ 1 · Первая живая кнопка (всплывашка)

В любое видимое место страницы добавляем кнопку:

```html
<button onclick="alert('Привет от кнопки!')">Нажми меня</button>
```

Сохранил (Ctrl+S) → обновил браузер (F5) → кликнул → выскочило окошко.

🎤 «`button` — кнопка. `onclick` — "при клике". `alert` — показать сообщение. Три слова — и кнопка живая.»

💬 **В чат:** «Какой текст бы вы вписали в свою всплывашку?»

> ⚠️ Кавычки только английские. Снаружи `"..."`, внутри `'...'` — не наоборот, иначе сломается.

---

## ШАГ 2 · Заводим `<script>` (место для действий)

В самый низ файла, перед `</body>`, добавляем пустой скрипт. Объясняем зачем — но коротко.

```html
<script>
  // сюда пишем действия
</script>
```

🎤 «Помните `<style>` — там жила красота. А `<script>` — тут живут действия. Кладём его ВНИЗ: код читается сверху вниз, и к этому моменту все кнопки уже на месте.»

> ⚠️ Это причина ошибки №1 у детей дома: скрипт наверху → код не находит кнопки. Проговори обязательно.

---

## ШАГ 3 · Счётчик — переменная (коробочка)

Сначала только заводим коробочку и показываем число.

```html
<h2 id="score">0</h2>
<button onclick="add()">+1</button>
```

В `<script>`:

```html
<script>
  let score = 0;
</script>
```

🎤 «`id="score"` — это имя элемента, чтобы JS его потом нашёл. `let score = 0` — коробочка по имени score, в ней лежит ноль.»

💬 **В чат:** «Что бы вы считали? Голы, уровни, лайки, рисунки?»

---

## ШАГ 4 · Счётчик оживает (кульминация урока)

Дописываем действие. Это главный момент «вау» — не торопись.

```html
<script>
  let score = 0;
  function add() {
    score = score + 1;
    document.getElementById('score').textContent = score;
  }
</script>
```

Кликаешь — число растёт. 🎉

🎤 «`function add()` — действие с именем. Кнопка зовёт его через `onclick="add()"`. `getElementById('score')` — найди элемент по имени. `textContent` — поменяй текст внутри. Каждый клик: +1 и обновить на экране.»

💬 **В чат:** «У кого в голове щёлкнуло, как это работает? 🔥»

> ⚠️ Имя `score` в `id` и в `getElementById` должно совпадать буква в букву. Вторая по частоте ошибка дома.

---

## ШАГ 5 · Кнопка красит (стиль по клику)

Возвращаемся к простому — пусть выдохнут после счётчика.

```html
<button onclick="document.body.style.background='#0f3460'">Тёмная</button>
<button onclick="document.body.style.background='#faf7f2'">Светлая</button>
```

Клик — фон меняется.

🎤 «JS умеет трогать любой стиль, как CSS — только по нажатию. `document.body` — вся страница, `.style.background` — её фон. Дома подставите свои цвета.»

💬 **В чат:** «Какие два цвета поставили бы на свои кнопки?»

---

## ШАГ 6 · (если есть время / для сильных) кнопка «−1» и «Сброс»

Не обязательный шаг. Если группа тянет — показываем, что функций может быть несколько.

```html
<button onclick="sub()">−1</button>
<button onclick="reset()">Сброс</button>

<script>
  function sub() {
    score = score - 1;
    document.getElementById('score').textContent = score;
  }
  function reset() {
    score = 0;
    document.getElementById('score').textContent = score;
  }
</script>
```

🎤 «Видите повтор? Одни и те же две строки. Позже научимся не повторять. Пока — просто копируем, это нормально.»

> 💡 Если кто-то в чате скучает («я знаю JS») — вот его задача: «сделай счётчик, который и прибавляет, и убавляет, и не уходит ниже нуля».

- Страничка урока 1 + заголовок-счётчик `0`
- Кнопка `+1` увеличивает число
- Кнопка-всплывашка с `alert`
- Две кнопки меняют фон
- (опц.) кнопки `−1` и «Сброс»

Финальная сверка — открой `index-final.html`, должно совпадать по поведению.

---

## 🆘 Если живой кодинг встал

1. **Скрипт «не работает»** → 90% случаев `<script>` выше элементов. Перенеси вниз.
2. **Совсем сломалось** → открой `index-final.html`, дальше показывай по нему. Не чини в прямом эфире у одного ребёнка.
3. **«Ошибка»** → F12 → Console, красная строка = номер строки с ошибкой. Но детям не углубляйся, просто «запишите, разберём по шпаргалке».
