# 📖 Словарик урока 2 — «Оживляем сайт»

> ⚠️ **ЧЕРНОВИК** (заготовка автозадачи от 6 июня 2026). Проверь формулировки под свою группу, потом убери плашку.
>
> Объясняем простыми словами, без «функция/событие/DOM/область видимости». Эти слова — позже.

| Слово | Простыми словами | Пример |
|---|---|---|
| **JavaScript (JS)** | Третий язык. HTML строит, CSS красит, **JS заставляет действовать**. | Кнопка, которая считает клики |
| **`<script>`** | Место, где живут действия. Как `<style>` для красоты — только для дела. Кладём **в самый низ** страницы. | `<script> ... </script>` |
| **кнопка `<button>`** | Кнопка на странице. Текст между тегами — надпись на ней. | `<button>Жми</button>` |
| **`onclick`** | «При клике». Внутри кавычек — что сделать, когда нажмут. | `onclick="add()"` |
| **`alert('...')`** | Всплывашка — окошко с сообщением. | `alert('Привет!')` |
| **переменная** | Коробочка с именем, в которой лежит значение. Можно менять. | `let score = 0` |
| **`let`** | Слово, которым заводят новую коробочку-переменную. | `let count = 0` |
| **действие с именем** | Кусочек кода со своим именем. Кнопка зовёт его по имени со скобками. (В учебниках — «функция», детям пока не нужно.) | `function add() { ... }` |
| **`id`** | Имя элемента, чтобы JS смог его найти. Придумываешь сам. | `<h2 id="score">` |
| **`getElementById`** | «Найди элемент по имени `id`». | `document.getElementById('score')` |
| **`textContent`** | Текст внутри элемента. Через него меняем, что написано. | `.textContent = score` |
| **`document.body`** | Вся страница целиком. Как `body` в CSS. | `document.body.style...` |
| **`.style`** | Доступ к стилям элемента из JS — как CSS, только по нажатию. | `.style.background = '#000'` |
| **Console (F12)** | Окно браузера, где видно ошибки. Красная строка = где сломалось. | F12 → вкладка Console |

---

## Три языка одной строкой

- **HTML** — *что показать* (теги: заголовок, картинка, кнопка)
- **CSS** — *как выглядит* (цвет, размер, форма)
- **JavaScript** — *что происходит при действии* (клик → счётчик растёт)

## Две ошибки, из-за которых «не работает»

1. `<script>` стоит **наверху** → код не находит кнопки. Лечится переносом вниз.
2. Имя в `id="score"` и в `getElementById('score')` **не совпадает** буква в букву.
