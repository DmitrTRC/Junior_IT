01Три языка — три работы

HTML
Строит. Теги — что показать (заголовок, картинка, кнопка).
CSS
Красит. Стили — какого цвета, размера, формы.
JavaScript
Оживляет. Действия — что произойдёт, когда нажмёшь.
💡
JS пишется внутри тега <script>. Клади его в самый низ страницы, перед </body> — иначе код не найдёт кнопки.

02Живая кнопка

Самое простое — кнопка, которая показывает сообщение.

<button onclick="alert('Привет!')">Нажми меня</button>
<button>
Кнопка. Текст между тегами — надпись на ней.
onclick
«При клике». Внутри кавычек — что сделать по нажатию.
alert('...')
Всплывашка. Показывает окошко с текстом.

03Счётчик (главное за урок)

Считаем, сколько раз нажали на кнопку.

<h2 id="score">0</h2> <button onclick="add()">+1</button> <script> let score = 0; function add() { score = score + 1; document.getElementById('score').textContent = score; } </script>
id="score"
Имя элемента. Чтобы JS его нашёл. Имя придумываешь сам.
let score = 0
Переменная — коробочка со значением. Здесь начинаем с нуля.
function add()
Действие с именем. Кнопка зовёт его через onclick="add()".
getElementById
«Найди по имени». Берёт элемент с нужным id.
.textContent
Поменять текст внутри элемента.

04Меняем стиль по клику

JS может трогать любой стиль — как CSS, только по нажатию.

// сменить фон страницы <button onclick="document.body.style.background='#0f3460'">Тёмная</button> <button onclick="document.body.style.background='#faf7f2'">Светлая</button>
document.body
Вся страница. Как body в CSS.
.style.background
Меняем фон. Так же: .style.color, .style.fontSize.

05Топ-5 ошибок новичка

1. <script> наверху. Клади его ВНИЗ, перед </body>. Иначе кнопок ещё нет, и код их не находит.
2. Разные имена. id="score" и getElementById('score') должны совпадать буква в букву.
3. Кавычки. Только английские ' ' или " ", не «ёлочки».
4. Забыл скобки у действия. Зовём add() со скобками, не просто add.
5. Не сохранил / не обновил. Ctrl+S, потом F5 в браузере.
🛠
Что-то не работает? Нажми F12 → вкладка Console. Красная строка покажет, в какой строке ошибка.
