Твой первый сайт

Сделать самому, сегодня вечером. Шпаргалка под рукой, всё получится.

1Поставь VS Code

Зайди на сайт code.visualstudio.com. Нажми большую синюю кнопку — она сама поймёт какая у тебя система (Windows или Mac). Скачай, установи. 5 минут максимум.

Зачем именно VS Code? Он подсвечивает теги цветом, подсказывает ошибки и автоматически закрывает теги. Блокнот всё это не умеет.

2Создай файл и папку

На рабочем столе создай папку — назови moi-sait (только английские буквы!).

Внутри папки создай файл index.html. Это имя стандартное — так называется главный файл любого сайта.

Открой этот файл в VS Code.

3Скопируй стартер

Это базовый код. Скопируй его в свой index.html и дальше будешь менять под себя:

<style> body { background: #1a1a2e; color: white; font-family: Arial, sans-serif; padding: 40px; } h1 { color: #ffd60a; font-size: 50px; text-align: center; } img { width: 300px; border-radius: 20px; display: block; margin: 20px auto; } a { color: #ffd60a; text-decoration: none; background: #16213e; padding: 12px 24px; border-radius: 8px; display: inline-block; transition: 0.3s; } a:hover { background: #ffd60a; color: #1a1a2e; transform: scale(1.1); } </style> <h1>Привет! Я ___</h1> <p>Мне ___ лет. Я ___.</p> <img src="ВСТАВЬ_СВОЮ_КАРТИНКУ"> <h2>Что я люблю:</h2> <ul> <li>___</li> <li>___</li> <li>___</li> </ul> <a href="https://...">Мой любимый сайт</a>

4Заполни своими данными

  1. Замени имя в <h1>
  2. Напиши свой возраст и хобби в <p>
  3. Найди свою картинку на unsplash.com и вставь её URL в src
  4. Напиши минимум 5 пунктов в <ul> — что любишь
  5. Вставь ссылку на свой любимый сайт в <a>

5Сделай это СВОИМ

Это самое главное. Поменяй цвета. Шрифт. Размеры. Поэкспериментируй.

✦ Идеи для прокачки

  • Поменяй цвета через coolors.co
  • Поставь border-radius: 50% на картинку — будет круглая
  • Добавь градиент: background: linear-gradient(135deg, цвет1, цвет2)
  • Добавь эмодзи в заголовок
  • Сделай несколько разных кнопок-ссылок
  • Поменяй шрифт через fonts.google.com (это уже продвинуто, но попробуй)

6Покажи и пришли

Когда сайт готов:

  1. Покажи родителям — заслуженно похвастайся
  2. Сделай скриншот (Print Screen на Windows, Cmd+Shift+4 на Mac)
  3. Пришли скриншот мне в чат — посмотрю, дам совет
  4. На следующий урок принеси файл — будем добавлять анимацию

✓ Проверь себя

Если открываешь свой index.html в браузере, и появляется страница с твоим именем, картинкой, списком и кнопкой — задача выполнена. Молодец.

Застрял?

Это нормально. Перечитай шпаргалку, проверь:

→ Все теги закрыты?
→ Кавычки английские " а не «ёлочки»?
→ Файл сохранён? (Ctrl+S)
→ Браузер обновлён? (F5)

Если совсем не получается — напиши мне, разберёмся вместе.
