# Урок 1 — Первый сайт за 75 минут

**Тема:** Личная страничка на HTML + CSS  
**Длительность:** 75 минут  
**Возраст:** 11–14 лет  
**Уровень:** с нуля

## Что ребёнок умеет к концу урока

- Понимает, что сайт — это файл с расширением `.html`
- Знает основные теги: `h1`, `h2`, `p`, `img`, `a`, `ul`, `li`
- Понимает что такое CSS и где он пишется (`<style>`)
- Умеет менять цвета, шрифты, отступы, рамки
- Знает что такое `:hover` и `transition`
- Может сделать страничку про себя дома по шпаргалке

## Порядок подготовки

1. За день — отправить родителям ссылку и [гайд подготовки устройств](../../docs/setup-guide.html)
2. За час — открыть все 3 демки, слайды, VS Code, пустую папку для живого кодинга
3. Чек-лист готовности — в конце [`scenario.md`](./scenario.md)

## После урока

1. Отправить в чат: шпаргалку, домашку, [словарик](./glossary.md)
2. Дать ссылку на [Soft vs Hard Skills](../../docs/soft-vs-hard-skills.html) — для тех, кто спросит «зачем вообще»
3. Собрать скриншоты домашек, дать каждому персональный фидбек
