Всего 3 концепции

01

Сайт — это файл

Обычный текстовый файл с расширением .html. Браузер его читает и показывает.

02

Внутри — теги

Команды браузеру, что показать. Как бутерброд:

<тег> что-то </тег>
03

У тегов есть имена

Каждое имя — свой результат. h1, p, img, a...
