## 🌍 Самое главное

**HTML** *(эйч-ти-эм-эл)* — язык, на котором написаны ВСЕ сайты в интернете. Если бы интернет был городом, HTML — это кирпичи и доски, из которых строят дома (страницы).

**CSS** *(си-эс-эс)* — язык, который красит и наряжает HTML. HTML говорит «здесь будет заголовок», CSS говорит «и пусть он будет жёлтый, по центру, большой».
> 🧠 HTML — это голый каркас квартиры (стены, окна, двери). CSS — обои, краска, мебель.

**Браузер** — программа, которая открывает сайты. Chrome, Safari, Edge, Яндекс.Браузер, Firefox — все это браузеры. Они читают `.html`-файл и превращают код в красивую страницу.

**Файл** — кусочек информации, сохранённый на компьютере. У файла есть **имя** и **расширение**.

**Расширение** — последние буквы после точки в имени файла. Они говорят компьютеру, что это за файл. `.html` — это сайт. `.jpg` — картинка. `.mp3` — музыка.

**`index.html`** — стандартное имя главной страницы сайта. Если файл назвать так — браузер откроет его автоматически.

## 🧱 Кирпичики страницы

**Тег** — команда браузеру, что показать. Выглядит как слово в угловых скобках: `<тег>`.
> 🧠 Тег — это как кнопка на пульте от телика. Нажал `<h1>` — получил заголовок. Нажал `<img>` — получил картинку.

**Открывающий и закрывающий теги** — большинство тегов работают парой:
```html
<h1>Привет</h1>
```
`<h1>` открывает заголовок, `</h1>` (со слэшем) закрывает. Между ними — содержимое.
> 🧠 Как бутерброд: хлеб сверху, начинка, хлеб снизу.

**Одиночный тег** — некоторые теги не закрываются, потому что у них нет «начинки». Например `<img>`, `<br>`, `<hr>`.

**Атрибут** — дополнительная настройка тега. Пишется внутри открывающего тега:
```html
<img src="моя_кошка.jpg">
```
Здесь `src` — атрибут, `"моя_кошка.jpg"` — его значение.
> 🧠 Тег — это машина. Атрибуты — цвет, колёса, навигатор. Машина одна, а настроек много.

**Элемент** — тег + всё что внутри него. `<h1>Привет</h1>` целиком — это элемент.

**Вложенность** — теги внутри тегов. Это нормально и нужно:
```html
<ul>
  <li>Пункт 1</li>
  <li>Пункт 2</li>
</ul>
```
`li` живут внутри `ul`. Это называется «`li` — потомок `ul`», «`ul` — родитель `li`».

## 🏷 HTML-теги, которые мы видели

**`<h1>` ... `<h6>`** — заголовки. `h1` — самый большой, `h6` — самый маленький. Обычно на странице один `h1`.

**`<p>`** — абзац (от англ. *paragraph*). Просто текст.

**`<img>`** — картинка (*image*). Одиночный тег. Без атрибута `src` бесполезен.

**`<a>`** — ссылка (*anchor* — «якорь»). Кликабельный текст, который ведёт куда-то. Без атрибута `href` ведёт в никуда.

**`<ul>`** — список с маркерами (точечки, *unordered list*).

**`<ol>`** — список с цифрами (*ordered list*).

**`<li>`** — пункт списка (*list item*). Живёт только внутри `<ul>` или `<ol>`.

**`<br>`** — перенос строки. Заставляет текст продолжиться с новой строки. Одиночный.

**`<hr>`** — горизонтальная линия-разделитель. Одиночный.

**`<style>`** — место, где живёт CSS. Пишется в самом верху файла.

---

## ⚙️ Атрибуты

**`src`** *(source — источник)* — адрес картинки в `<img>`. Можно ссылку из интернета, можно имя файла рядом.

**`href`** *(hypertext reference — ссылка)* — куда ведёт `<a>`. Обычно полный URL: `href="https://youtube.com"`.

**`alt`** *(alternative — альтернатива)* — текст, который покажется вместо картинки, если она не загрузится. Также его читают вслух программы для слабовидящих.

## 🎨 CSS-свойства

CSS-правило выглядит так:
```css
имя_тега {
  свойство: значение;
}
```

**`background`** — фон элемента. Может быть цветом или градиентом.
```css
background: #1a1a2e;
background: linear-gradient(135deg, red, blue);
```

**`color`** — цвет ТЕКСТА. Не путать с background.

**`font-family`** — какой шрифт использовать. `Arial`, `Times New Roman`, `Inter`, etc.

**`font-size`** — размер текста, обычно в пикселях: `font-size: 24px`.

**`text-align`** — куда выровнять текст: `left`, `center`, `right`.

**`padding`** — отступ ВНУТРИ блока, от рамки до содержимого.
> 🧠 Это как пенопласт внутри коробки с пиццей. Сама пицца не у стенок коробки, а в серединке.

**`margin`** — отступ СНАРУЖИ блока, от него до соседей.
> 🧠 Это расстояние между двумя коробками с пиццей на столе.

**`border`** — рамка вокруг блока. Пишем толщину, тип, цвет:
```css
border: 4px solid yellow;
```

**`border-radius`** — закругление углов. `50%` — получится круг.

**`width`** / **`height`** — ширина и высота блока. Чаще всего для картинок.

## ✨ Магия (то, что выглядит «по-взрослому»)

**`:hover`** *(хо́вер — «наведение»)* — что произойдёт, когда мышка наведётся на элемент.
```css
a:hover { background: red; }
```
Эта строчка значит: «когда мышка над ссылкой — сделай фон красным». Уберёшь мышку — всё вернётся.

**`transition`** *(транзи́шн — «переход»)* — плавность изменения. Без него `:hover` срабатывает мгновенно (резко). С ним — плавно за указанное время.
```css
transition: 0.3s;
```
0.3 секунды — стандарт, выглядит мягко.

**`transform`** *(трансфо́рм — «преобразование»)* — поворот, масштаб, сдвиг.
```css
transform: scale(1.1);   /* увеличить на 10% */
transform: rotate(5deg); /* повернуть на 5 градусов */
```

**`linear-gradient`** *(лине́арный градие́нт)* — плавный переход цвета. Не один цвет, а несколько с переходами.
```css
background: linear-gradient(135deg, #1a1a2e, #16213e);
```
`135deg` — угол наклона (в градусах). Дальше — цвета через запятую.

## 🔧 Инструменты и форматы

**VS Code** *(ви-эс код)* — программа, в которой пишут код. Подсвечивает теги цветами, подсказывает, ругается на ошибки. Бесплатная, от Microsoft.

**Live Server** — расширение для VS Code. После сохранения файла страница в браузере обновляется сама — не надо жать F5.

**HEX** *(хекс)* — способ записи цвета через решётку и 6 символов: `#ffd60a`. Это шестнадцатеричный код красного-зелёного-синего.
> 🧠 `#FF0000` — чистый красный (FF красного = максимум, остальные = 0). `#000000` — чёрный. `#FFFFFF` — белый.

**px** *(пиксель)* — единица измерения. Один пиксель — одна точечка на экране. Используется почти везде: `font-size: 24px`, `padding: 40px`.

**%** — процент от размера родителя. `width: 50%` значит «половина ширины контейнера, в котором я нахожусь».

## 💡 Дополнительно

**Unsplash, Pexels** — бесплатные сайты с фотографиями. Можно брать любые картинки оттуда в свои сайты.

**coolors.co** — сайт с готовыми сочетаниями цветов. Если не знаешь, какие цвета подойдут — иди туда, листай палитры, копируй HEX-коды.

**MDN** *(developer.mozilla.org)* — самая честная и полная документация по веб-разработке. На русском есть. Когда понадобится узнать про какой-нибудь тег — гугли «mdn тег» и читай.

## 🚫 Что НЕ объяснял на уроке (специально)

Эти слова пока не нужны, но услышишь — не пугайся:

- **DOCTYPE** — техническая строка в самом верху файла. Скоро узнаешь.
- **DOM** — то, как браузер видит твою HTML-страницу внутри себя.
- **семантика** — почему в одних местах используют `<h1>`, а в других `<p>` — это про смысл, а не про вид.
- **отзывчивость / адаптив** — когда сайт по-разному выглядит на телефоне и компьютере.
- **JavaScript** — третий язык веба. Делает сайты живыми. Будем на следующем уроке.

Эти слова — обещание следующих уроков, не повод грузиться сейчас.
