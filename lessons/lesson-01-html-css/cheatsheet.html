Всё с первого урока, на одной странице

01Скелет любого сайта

Просто скопируй это в файл index.html и начинай менять.

<!-- index.html --> <style> body { background: #1a1a2e; color: white; padding: 40px; } h1 { color: #ffd60a; } </style> <h1>Заголовок</h1> <p>Абзац текста.</p> <img src="https://...">

02HTML-теги

<h1>...</h1>
Главный заголовок. Самый большой текст на странице. Обычно один.
<h2>...</h2>
Подзаголовок. Поменьше. Их может быть несколько.
<p>...</p>
Абзац. Обычный текст.
<img src="...">
Картинка. Внутри `src` — адрес картинки. Тег закрывать не нужно.
<img src="https://images.unsplash.com/photo-...">
<a href="...">
Ссылка. Внутри `href` — адрес куда вести.
<a href="https://youtube.com">Мой ютуб</a>
<ul><li>
Список. `ul` снаружи, `li` — каждый пункт внутри.
<br>
Перенос строки. Не закрывается. Используй когда хочешь начать с новой строки внутри абзаца.
<hr>
Горизонтальная черта. Разделитель между блоками.

03CSS-свойства

CSS пишется внутри <style>...</style> в начале файла. Правила выглядят так:

имя_тега { свойство: значение; свойство: значение; }
background
Фон. Цвет или градиент.
background: #1a1a2e;
background: linear-gradient(135deg, red, blue);
color
Цвет текста.
color: white;
font-size
Размер текста. В пикселях.
font-size: 24px;
font-family
Шрифт.
font-family: Arial, sans-serif;
text-align
Выравнивание текста: `left`, `center`, `right`.
padding
Отступ ВНУТРИ (от края рамки до содержимого).
padding: 20px;
margin
Отступ СНАРУЖИ (от блока до соседних).
border
Рамка. Толщина, тип, цвет.
border: 4px solid yellow;
border-radius
Закругление углов. Поставь `50%` для круга.
border-radius: 20px;
width / height
Ширина и высота. Часто для картинок.
width: 300px;
:hover
Магия! Что произойдёт когда наведёшь мышкой.
a:hover { background: red; }
transition
Плавное изменение. Поставь в правило, чтобы hover работал плавно.
transition: 0.3s;
transform
Поворот / масштаб.
transform: scale(1.1);
transform: rotate(5deg);
box-shadow
Тень. Размер X, размер Y, размытие, цвет.
box-shadow: 0 4px 20px rgba(0,0,0,0.3);

05Картинки — где брать

unsplash.com
Бесплатные фото. Найди картинку → правый клик → "Копировать адрес изображения" → вставь в `src`.
pexels.com
То же самое, тоже бесплатно.
⚠ Внимание: Если копируешь адрес страницы вместо адреса картинки — не сработает. Адрес картинки заканчивается на `.jpg`, `.png` или `.webp`.

06Эмодзи

Вставляй прямо в текст. Тегов не нужно.

Windows
Нажми Win + . (точка) — откроется панель эмодзи.
Mac
Нажми Ctrl + Cmd + пробел.
emojipedia.org
Просто погугли эмодзи, найди, скопируй, вставь.

07Топ-5 ошибок новичка

1. Забыл закрыть тег. Каждый `<тег>` нужно закрыть `</тег>`. Исключения: `<img>`, `<br>`, `<hr>`.
2. Перепутал кавычки. Используй обычные английские ", не «ёлочки» и не „лапки".
3. Забыл `;` в CSS. Каждое правило заканчивается точкой с запятой.
4. Скопировал не тот адрес картинки. Нужен прямой URL, заканчивающийся на `.jpg/.png/.webp`.
5. Не сохранил файл. Ctrl+S после каждого изменения. Без этого браузер не покажет новое.

08Что дальше

На следующем уроке: JavaScript — оживляем сайт. Кнопки начнут что-то делать, появятся счётчики, всплывающие окна, анимации.

Через 3-4 урока — выложим твою страничку в интернет по реальной ссылке. Будешь давать друзьям.
