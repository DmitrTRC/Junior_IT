# 💻 ЖИВОЙ КОД: что именно ты пишешь в эфире

Это последовательность из 8 шагов. На каждом шаге даю:
1. Что добавить.
2. Полный текущий файл (чтобы свериться).
3. Что говорить.

> **Совет:** распечатай этот файл или открой на втором экране/телефоне. В прямом эфире лезть в Markdown через шаринг — плохо.

---

## ШАГ 1: Самое начало

🎤 «Создаю файл index.html. Пишу первую строку.»

```html
<h1>Привет! Я Артём</h1>
```

**Сохрани (Ctrl+S) и открой в браузере двойным кликом.**

Покажи: огромный заголовок «Привет! Я Артём».

🎤 «Один тег — один результат. Видите как просто?»

---

## ШАГ 2: Добавляем абзац

🎤 «Теперь добавлю абзац — обычный текст.»

```html
<h1>Привет! Я Артём</h1>
<p>Мне 12 лет. Я люблю Майнкрафт и собаку.</p>
```

**Сохрани, обнови браузер (F5).**

🎤 «Видите разницу? `h1` крупный и жирный — это заголовок. `p` обычный — это абзац.»

---

## ШАГ 3: Картинка

🎤 «Самое волшебное — картинка. Это надо видеть.»

```html
<h1>Привет! Я Артём</h1>
<p>Мне 12 лет. Я люблю Майнкрафт и собаку.</p>

<img src="https://images.unsplash.com/photo-1587573089734-09cb69c0f2b4?w=400" alt="моя собака">
```

**Сохрани, обнови.**

🎤 «Появилась собака. `img` — особый тег: он не закрывается. И у него есть `src` — это адрес картинки в интернете.»

---

## ШАГ 4: Список любимого

```html
<h1>Привет! Я Артём</h1>
<p>Мне 12 лет. Я люблю Майнкрафт и собаку.</p>

<img src="https://images.unsplash.com/photo-1587573089734-09cb69c0f2b4?w=400" alt="моя собака">

<h2>Что я люблю:</h2>
<ul>
  <li>Майнкрафт</li>
  <li>Пиццу с колбасой</li>
  <li>Свою собаку Рекса</li>
  <li>Гонять на велике</li>
</ul>
```

**Сохрани, обнови.**

🎤 «`h2` — заголовок поменьше. `ul` — список. `li` — пункт списка. Внутри `ul` могут быть только `li` — это правило.»

## ШАГ 6: Первые стили

🎤 «Теперь самое интересное. Добавляю в самое начало файла тег style — это место для CSS.»

```html
<style>
  body {
    background: #1a1a2e;
    color: white;
    font-family: Arial, sans-serif;
    padding: 40px;
  }
</style>

<h1>Привет! Я Артём</h1>
<p>Мне 12 лет. Я люблю Майнкрафт и собаку.</p>

<img src="https://images.unsplash.com/photo-1587573089734-09cb69c0f2b4?w=400" alt="моя собака">

<h2>Что я люблю:</h2>
<ul>
  <li>Майнкрафт</li>
  <li>Пиццу с колбасой</li>
  <li>Свою собаку Рекса</li>
  <li>Гонять на велике</li>
</ul>

<a href="https://minecraft.net">Мой любимый сайт</a>
```

**Сохрани, обнови. Страница превращается в тёмную с белым текстом.**

🎤 «Магия. Одно правило для `body` — поменялась вся страница. Потому что `body` — это вся страница.»

## ШАГ 7: Стилизуем заголовок и картинку

```html
<style>
  body {
    background: #1a1a2e;
    color: white;
    font-family: Arial, sans-serif;
    padding: 40px;
  }

  h1 {
    color: #ffd60a;
    font-size: 60px;
    text-align: center;
  }

  img {
    width: 300px;
    border-radius: 20px;
    border: 4px solid #ffd60a;
  }
</style>
```

**Сохрани, обнови.**

🎤 «Заголовок жёлтый и огромный. Картинка с рамкой и закруглёнными углами. Каждое правило — для своего тега.»
