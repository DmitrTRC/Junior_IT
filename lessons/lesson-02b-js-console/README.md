# Урок 2.5 — Сам язык: консоль-калькулятор

**Тема:** Консоль браузера как калькулятор. Числа, переменные, текст, `prompt`/`alert`. Без файлов и без DOM.
**Длительность:** 60–75 минут
**Возраст:** 11–14 лет
**Место в курсе:** мост между уроком 2 (первые кнопки) и уроком 3 (события/анимация).

## Зачем вставлен этот урок

На уроке 2 счётчик через `document.getElementById('score').textContent = score` оказался стеной: сразу слишком много «магии» (объект, метод, свойство, цепочка через точку). Дети залипли именно тут. Этот урок откатывается к **самому языку** — считать, хранить в переменных, спрашивать и показывать — целиком в консоли браузера, без единого обращения к странице. Кнопки вернём на уроке 3, и тогда `getElementById` ляжет на готовую почву.

## Что ребёнок умеет к концу урока

- Открывает консоль браузера (F12 → Console) и понимает, что это «калькулятор, который всегда под рукой»
- Считает выражения: `+ - * /`, скобки, проценты
- Заводит переменную-коробочку через `let` и перезаписывает её
- Различает число и текст (строку); понимает ловушку `"5" + 5`
- Спрашивает через `prompt`, показывает через `alert`, печатает через `console.log`
- Превращает текст в число через `Number(...)`
- Может дома собрать мини-программу «спросить → посчитать → показать» по шпаргалке

## Связь с соседними уроками

- **Назад (урок 2):** «помните длинную команду счётчика? Сегодня — из чего она состоит».
- **Вперёд (урок 3):** этот же язык повесим на кнопки страницы — и счётчик станет понятным.
