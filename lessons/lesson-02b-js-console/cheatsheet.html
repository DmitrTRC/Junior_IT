01Где печатать: консоль браузера

Тебе не нужны файлы и кнопки. В любом браузере уже есть место, где можно писать на JavaScript и сразу видеть ответ. Это консоль.

F12
Открыть консоль (Windows). Дальше — вкладка Console.
Cmd + Opt + J
Открыть консоль (Mac, Chrome). Сразу попадаешь в Console.
Enter
Печатаешь строку → жмёшь Enter → браузер отвечает. Как чат, только с компьютером.
💡
Консоль есть всегда — на любом сайте, даже без интернета. Открыл браузер — у тебя под рукой калькулятор, который умеет гораздо больше «плюс-минус».

👀Почему серый «undefined»?

После каждой строки консоль серым (со стрелкой ‹) показывает итог — что строка «вернула». У некоторых строк итога нет — тогда она честно пишет undefined («ничего»). Это не ошибка.

let a = 4;undefinedconsole.info("a = " + a)a = 4undefined
let a = 4
Завёл коробочку. Итога-значения нет → серое undefined. Норма.
console.info(...)
Сначала печатает (белая строка a = 4), а возвращает «ничего» → снова серое undefined.
✅
Хочешь «настоящий» итог — введи выражение со значением: набери a или 2 + 2 и нажми Enter. Серым будет уже 4.

02Считает что угодно

Просто пиши пример и жми Enter.

2 + 24100 / 714.2857...(5 + 3) * 1080// сколько это 15% от 1500?1500 * 0.15225
+ - * /
Плюс, минус, умножить, делить. Звёздочка * — это «умножить», слэш / — «делить».
( )
Скобки работают как в математике — сначала считается то, что внутри.

03Переменная — коробочка с именем

Чтобы не считать одно и то же дважды, кладём число в коробочку и даём ей имя.

// завели коробочку cena, положили 250let cena = 250// теперь имя = значениеcena * 3750// можно перезаписатьcena = 300cena * 3900
let
Заводим новую коробочку. Слово let нужно только один раз — когда создаём.
cena = 300
Перезаписать. Без let — просто кладём в коробочку новое значение.
имя
Придумываешь сам, английскими буквами, без пробелов: ochki, vozrast, summa.

04Число и текст — разные вещи

Число считается. Текст (его называют строка) — пишется в кавычках и «складывается» как слова.

// текст — в кавычкахlet imya = "Артём""Привет, " + imya"Привет, Артём"// осторожно: число в кавычках — уже текст!"5" + 5"55" // склеилось, а не сложилось
"текст"
Строка. Всё в кавычках — это текст, даже "5". С текстом + — это «склеить».
5
Число. Без кавычек. С числами + — это «сложить».

05Спроси и покажи

Три команды, чтобы программа «поговорила» с человеком.

// спросить (ответ всегда ТЕКСТ)let name = prompt("Как тебя зовут?")// показать окошкоalert("Привет, " + name + "!")// напечатать в консольconsole.log("Готово")
prompt("...")
Спросить. Показывает окошко с полем ввода. Ответ — всегда текст.
Number(...)
Сделать числом. Если спросил число — оберни: Number(prompt("Сколько лет?")).
alert("...")
Показать сообщение в окошке.
console.log(...)
Напечатать прямо в консоль. Удобно подсмотреть, что лежит в коробочке.

06Мини-программа в консоли

Спросили возраст — посчитали год рождения. Всё в консоли, без единой кнопки.

let vozrast = Number(prompt("Сколько тебе лет?"))let god = 2026 - vozrastalert("Ты родился примерно в " + god)
🧮
Так же считается что угодно своё: чаевые (summa * 0.1), средний балл, сколько секунд в твоём возрасте. Калькулятор, который ты программируешь сам.

07Топ-5 ошибок новичка

1. Кавычки-«ёлочки». Только английские " или ', не « и не ». Иначе красная ошибка.
2. Число в кавычках. "5" + 1 даёт "51", а не 6. Для счёта — без кавычек.
3. Забыл Number у prompt. prompt возвращает текст — заверни в Number(...), если надо считать.
4. Второй раз пишу let. let — один раз при создании. Меняешь значение — без let.
5. Русская раскладка. Команды пишутся английскими буквами. лет ≠ let.
🛠
Красная строка в консоли — это не страшно. Она прямо говорит, что не так. Прочитай слово после слова Error — обычно там и ответ.
