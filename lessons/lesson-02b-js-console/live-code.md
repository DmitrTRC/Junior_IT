# 💻 LIVE-CODE: «Сам язык — консоль-калькулятор» (урок 2.5)

**Стартовая точка:** НИКАКИХ файлов. Открываешь браузер → консоль (F12 → Console). Всё пишем прямо там.
**Зачем этот урок:** на уроке 2 счётчик через `document.getElementById('score').textContent` оказался стеной — слишком длинная «магия» сразу. Сегодня откатываемся к самому языку: считать, хранить, спрашивать. Без страницы, без DOM. Кнопки вернём на следующем уроке — и тогда они станут понятными.
**Принцип:** одна строка = один ответ браузера. Между шагами — пинг в чат. Теории не дольше 2 минут подряд.

---

## ШАГ 0 · Консоль — это калькулятор, который всегда с тобой

🎬 Открываешь любую вкладку → `F12` → вкладка **Console**. Курсор в строке после `>`.

🎤 «Помните, в прошлый раз кнопка-счётчик была сложной — длинная команда, которую тяжело понять. Сегодня я покажу, откуда всё растёт. И нам не нужны ни файлы, ни кнопки — только вот это окошко. Оно есть в любом браузере, на любом сайте, даже без интернета.»

🎬 Печатаешь `2 + 2`, Enter → `4`.

🎤 «Это калькулятор. Но не простой — сейчас увидите, насколько.»

💬 **В чат:** «Откройте F12 дома потом — проверим, у всех ли открылась консоль. Сейчас просто смотрите.»

> ⚠️ Покажи, КУДА именно печатать (вкладка Console, не Elements). Самый частый «у меня не работает» дома — печатают не туда.

## ШАГ 1 · Считает что угодно

🎬 Набираешь по очереди, проговаривая:

```js
100 / 7
(5 + 3) * 10
1500 * 0.15      // 15% от 1500
```

🎤 «Звёздочка — умножить, слэш — делить. Скобки — как в математике. `1500 * 0.15` — это 15 процентов от 1500, смотрите: 225. Реальная задачка, а не учебная.»

💬 **В чат:** «Какой пример посчитать? Кидайте числа — посчитаю вживую.»

> 💡 Возьми 1–2 числа из чата и посчитай прямо в консоли. Дети любят, когда считают «их» пример.

## ШАГ 2 · Переменная — коробочка с именем

🎤 «Чтобы не набирать число каждый раз, кладём его в коробочку и даём имя.»

```js
let cena = 250
cena * 3
```

🎬 Ответ `750`.

🎤 «`let` — "заведи коробочку". `cena` — имя, придумал сам. Теперь вместо числа можно писать имя. Поменяем цену:»

```js
cena = 300
cena * 3
```

🎤 «Заметьте: второй раз `let` НЕ пишу. `let` — только когда создаём. Дальше просто кладём новое.»

💬 **В чат:** «Как назвали бы свою коробочку? ochki, goly, lvl?»

> ⚠️ Имя — английскими буквами, без пробелов. Это грабля №1 дома (`моя цена` сломается).

---

## ШАГ 3 · Число и текст — две разные вещи

🎤 «Теперь важное. Браузер различает ЧИСЛА и ТЕКСТ. Текст пишем в кавычках.»

```js
let imya = "Артём"
"Привет, " + imya
```

🎬 Ответ `"Привет, Артём"`.

🎤 «С текстом плюс — это "склеить". А теперь ловушка, смотрите внимательно:»

```js
"5" + 5
```

🎬 Ответ `"55"`.

🎤 «Пятёрка в кавычках — это уже не число, а текст. И плюс её не сложил, а склеил: "55". Запомните: кавычки превращают что угодно в текст.»

💬 **В чат:** «🤯 если ловушка с "5" + 5 удивила.»

> ⚠️ Это ядро будущих ошибок. Проговори медленно — здесь «щёлкает» понимание, зачем нужен `Number(...)` на следующем шаге.

## ШАГ 4 · Спроси и покажи: prompt / alert / console.log

🎤 «Калькулятор умеет разговаривать. Три команды.»

```js
let name = prompt("Как тебя зовут?")
alert("Привет, " + name + "!")
console.log("name =", name)
```

🎬 Вводишь имя в окошко → выскакивает приветствие → в консоли печатается значение.

🎤 «`prompt` — спросить, показывает окошко. `alert` — показать сообщение. `console.log` — напечатать в консоль, чтобы подсмотреть, что внутри. ВАЖНО: `prompt` всегда возвращает ТЕКСТ — даже если ввёл число.»

💬 **В чат:** «Какой вопрос задали бы своей программе?»

> ⚠️ `prompt` блокирует страницу, пока не ответишь. Если «завис» — ищи окошко, оно могло открыться сверху.

---

## ШАГ 5 · Мини-программа (кульминация)

🎤 «Соберём настоящую маленькую программу. Спросим возраст — посчитаем год рождения.»

```js
let vozrast = Number(prompt("Сколько тебе лет?"))
let god = 2026 - vozrast
alert("Ты родился примерно в " + god)
```

🎬 Вводишь `12` → `Ты родился примерно в 2014`.

🎤 «Видите `Number(...)`? Это "сделай из текста число". Без него `2026 - "12"`... вот дома проверьте, что будет. Это и есть программа: спросили — посчитали — показали. Три строки.»

💬 **В чат:** «🔥 у кого получилось бы посчитать так свой год?»

> 💡 Если убрать `Number`, `2026 - "12"` всё равно даст 2014 (минус сам превращает текст в число), а вот `+` сломался бы: `2026 + "12"` склеит в `"202612"`. Не углубляйся вживую — оставь как «домашнюю загадку».

## ШАГ 6 · (если есть время / для сильных) свой калькулятор

🎤 «Кто хочет — вот калькулятор чаевых. 10% от счёта.»

```js
let summa = Number(prompt("Сумма счёта?"))
alert("Чаевые 10%: " + (summa * 0.1))
```

🎤 «Поменяйте 0.1 на 0.15 — будет 15%. Дома сделаете свой: средний балл, голы за сезон, что угодно.»

> 💡 Скучающему («я знаю JS») — задача: «спроси три оценки и посчитай среднее одним alert».

## ИТОГ на экране (что прозвучало)

- Консоль = калькулятор в любом браузере (`F12 → Console`)
- Считает: `+ - * /`, скобки, проценты
- `let имя = значение` — коробочка; `let` только при создании
- Текст в кавычках; `+` с текстом — склейка; `"5"` ≠ `5`
- `prompt` спросить · `alert` показать · `console.log` напечатать
- `Number(...)` — текст в число
- Мини-программа: спросить → посчитать → показать

## 🆘 Если живой кодинг встал

1. **«Красная ошибка»** → прочитай слово после `Error`. Чаще всего: кавычки-ёлочки или русская раскладка. Не чини у одного ребёнка — «запишите, разберём по шпаргалке».
2. **Печатают не туда** → ещё раз покажи: вкладка **Console**, строка после `>`.
3. **Совсем сломалось** → открой `index-final.html`, дальше веди по нему: там готовые строки с ответами.
4. **`prompt` «завис»** → окошко открылось, но не видно — сверни/разверни, оно блокирует до ответа.
