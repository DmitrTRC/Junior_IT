# 📖 Словарик урока 2.5 — «Сам язык: консоль-калькулятор»

> Объясняем простыми словами, без «функция/событие/DOM/метод/объект». Эти слова — позже. Сегодня только сам язык.

| Слово | Простыми словами | Пример |
|---|---|---|
| **Консоль** | Окошко в браузере, где можно писать на JS и сразу видеть ответ. Калькулятор, который всегда под рукой. | `F12` → вкладка **Console** |
| **F12 / Cmd+Opt+J** | Как открыть консоль: F12 на Windows, `Cmd+Opt+J` в Chrome на Mac. | — |
| **выражение** | То, что браузер посчитает: пример из чисел и знаков. | `(5 + 3) * 10` |
| **оператор** | Знак действия: сложить, вычесть, умножить, делить. | `+`  `-`  `*`  `/` |
| **переменная** | Коробочка с именем, в которой лежит значение. Можно перезаписать. | `let cena = 250` |
| **`let`** | Слово, которым **создают** новую коробочку. Нужно только один раз. | `let ochki = 0` |
| **число** | То, что считается. Пишется **без** кавычек. | `5`, `2026`, `0.15` |
| **строка (текст)** | Буквы и слова. Пишется **в кавычках**. С текстом `+` — это «склеить». | `"Привет"` |
| **кавычки** | Превращают что угодно в текст. Только английские `"` или `'`. | `"5"` — это уже текст |
| **`prompt('...')`** | Спросить у человека. Показывает окошко с полем. Ответ — **всегда текст**. | `prompt("Имя?")` |
| **`Number(...)`** | Превратить текст в число. Нужно для счёта после `prompt`. | `Number(prompt("Лет?"))` |
| **`alert('...')`** | Показать сообщение в окошке. | `alert("Готово")` |
| **`console.log(...)`** | Напечатать в консоль — подсмотреть, что внутри коробочки. | `console.log(ochki)` |
| **`undefined`** | «Ничего». Консоль пишет это серым, когда у строки нет итога-значения — например после `let` или `console.log`. **Не ошибка.** | `let a = 4` → серое `undefined` |
| **ошибка (Error)** | Красная строка в консоли. Не страшно — она прямо говорит, что не так. | читай слово после `Error` |

---

## Главная мысль урока одной строкой

Тебе **не нужны** ни файлы, ни кнопки, чтобы программировать. Открыл браузер → консоль → и язык уже работает: считает, помнит, спрашивает, отвечает.

## Две ловушки, из-за которых «не работает»

1. **Кавычки делают текст.** `"5" + 5` → `"55"` (склеилось), а не `10`. Для счёта — без кавычек.
2. **`prompt` отдаёт текст.** Чтобы считать ответ, оберни в `Number(...)`.
