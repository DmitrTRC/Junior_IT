4Спроси и покажи

let name = prompt("Как тебя зовут?")alert("Привет, " + name + "!")console.log("name =", name)
prompt всегда возвращает текст. alert — окошко. console.log — печать в консоль.

5Мини-программа (кульминация)

let vozrast = Number(prompt("Сколько тебе лет?"))let god = 2026 - vozrastalert("Ты родился примерно в " + god)
Number(...) превращает текст из prompt в число. Спросили → посчитали → показали.
