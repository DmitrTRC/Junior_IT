1Сравнение → да/нет

10 > 5true3 > 8false// «равно?» — три знака равно7 === 7true

2= положить · === сравнить (главная ловушка)

// один = кладёт в коробочкуlet ochki = 5// три === спрашиваетochki === 5trueochki === 10false
= «стало», === «равно ли?». Внутри if — три равно.

3if — сделать, только если да

let ball = 80if (ball >= 50) {  alert("Зачёт!")}// 80 >= 50 → true → выскочит. ball = 30 → тишина.

4if / else — две дороги

let ball = 30if (ball >= 50) {  alert("Зачёт! 🎉")} else {  alert("Почти — нужно 50")}

5Спроси и реши (кульминация)

let ball = Number(prompt("Сколько баллов набрал?"))if (ball >= 50) {  alert("Зачёт! 🎉")} else {  alert("Почти. Нужно 50, у тебя " + ball)}
Number обязателен: "100" > "9" → false (текст по буквам). Спросили → сравнили → решили.

6else if — несколько веток

let xp = Number(prompt("Сколько у тебя XP?"))if (xp >= 1000) { alert("🥇 Золото") }else if (xp >= 500) { alert("🥈 Серебро") }else { alert("🥉 Бронза — качайся дальше") }
Сверху вниз, по первой подошедшей ветке: 1500 → золото, 700 → серебро, 100 → бронза.

7Бонус: и / или

let age = Number(prompt("Сколько тебе лет?"))if (age >= 11 && age <= 14) {  alert("Ты как раз наш Junior!")} else {  alert("Курс для 11–14, но заходи :)")}
