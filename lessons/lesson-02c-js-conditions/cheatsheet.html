условия · if / else · урок 2.6

01Сравнение: ответ «да» или «нет»

Браузер умеет не только считать, но и сравнивать. В ответ — true (да) или false (нет). Это новый вид значения: не число и не текст.

10 > 5true3 > 8false// «равно?» — это ТРИ знака равно7 === 7true
> < >= <=
Больше, меньше, больше-или-равно, меньше-или-равно.
=== !==
Равно ли / не равно ли. Ответ всегда true или false.

⚠️Главная ловушка: = и === — РАЗНОЕ

Один знак равно кладёт в коробочку. Три знака равно сравнивают. Внутри if почти всегда нужно ===.

// один = положить (это НЕ вопрос)let ochki = 5// три === сравнить (это вопрос)ochki === 5trueochki === 10false
🔑
Запомни так: = это «стало» (положили), === это «равно ли?» (спросили). Перепутаешь в if — программа не спросит, а перезапишет коробочку.

02if — сделать, только если «да»

В круглых скобках — вопрос (условие). В фигурных — что делать, если ответ true. Если false — браузер просто пропустит.

let ball = 80if (ball >= 50) {  alert("Зачёт!")}// 80 >= 50 → true → выскочит «Зачёт!»
if ( условие )
Круглые скобки — вопрос с ответом да/нет.
{ ... }
Фигурные скобки — что делать, когда условие true.

03else — а иначе

Вторая дорога: что делать, если условие false. Программа всегда пойдёт ровно по одной из дорог.

let ball = 30if (ball >= 50) {  alert("Зачёт! 🎉")} else {  alert("Почти — нужно 50")}

04else if — ещё ветка

Дорог может быть больше двух. Браузер проверяет сверху вниз и идёт по первой подошедшей.

let xp = 700if (xp >= 1000) { alert("🥇 Золото") }else if (xp >= 500) { alert("🥈 Серебро") }else { alert("🥉 Бронза") }// 700 → серебро

05Спроси и реши

Соединяем с прошлым уроком: спросить через prompt, превратить в число через Number, сравнить и решить.

let ball = Number(prompt("Сколько баллов набрал?"))if (ball >= 50) {  alert("Зачёт! 🎉")} else {  alert("Почти. Нужно 50, у тебя " + ball)}
🧮
Так решается что угодно своё: ранг по очкам, зачёт по нормативу, успеешь ли до дедлайна. Программа сама выбирает ответ по твоим данным.

06И / ИЛИ (для сильных)

Можно проверить сразу два условия.

// && — «и» (оба верны)if (age >= 11 && age <= 14) { alert("Наш Junior!") }// || — «или» (хватит одного)if (den === "сб" || den === "вс") { alert("Выходной") }

07Топ-5 ошибок новичка

1. = вместо ===. Внутри if нужно три равно. Один = не спросит, а перезапишет коробочку.
2. Сравнил текст. Забыл Number(prompt(...)) → "100" > "9" даёт false (сравнило по буквам).
3. Забыл фигурные скобки. После if (...) идёт { ... } с действием.
4. Кавычки-«ёлочки». Только английские " или ', не « и не ».
5. Русская раскладка. if, else — английскими буквами. иф ≠ if.
🛠
Если ничего не происходит — первым делом проверь = / === внутри if. Это причина №1.
