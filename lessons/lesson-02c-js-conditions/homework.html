Два калькулятора,
которые решают

Сегодня соберёшь два настоящих калькулятора в консоли. Главное в обоих — if / else. Шпаргалка рядом.

1Открой консоль

Открой любой сайт (хоть пустую вкладку) и нажми F12 → вкладка Console. На Mac в Chrome — Cmd + Opt + J.

Печатаешь строку → Enter → браузер отвечает. Сегодня он будет считать и решать.

2Калькулятор ИМТ (вес и рост)

ИМТ — индекс массы тела. Его считают по простой формуле: вес поделить на рост в квадрате. А дальше программа сама скажет, в какую «полку» попало число.

Формула: ИМТ = вес / (рост × рост), где рост — в метрах.

💛
Сначала прочитай, потом запускай. Это тренажёр для if/else, а не приговор. У растущих ребят 11–14 лет ИМТ врачи смотрят по возрастным таблицам — взрослые цифры тут не работают. Считай для себя или для выдуманного героя/спортсмена — для тренировки кода без разницы. И не пугайся длинного хвоста после точки: округлять научимся позже.
// рост спросим в сантиметрах и поделим на 100 — получим метрыlet ves = Number(prompt("Вес в кг?"))let rost = Number(prompt("Рост в сантиметрах? (например 155)")) / 100let bmi = ves / (rost * rost)if (bmi < 18.5) {  alert("ИМТ " + bmi + " — ниже нормы")} else if (bmi < 25) {  alert("ИМТ " + bmi + " — норма 👍")} else {  alert("ИМТ " + bmi + " — выше нормы")}

Здесь три ветки через if → else if → else. Программа проверяет сверху вниз и идёт по первой подошедшей.

3Хватит ли денег на покупку

Второй калькулятор — из обычной жизни. Спросим, сколько денег есть и сколько стоит вещь, и программа решит: брать или копить.

let deneg = Number(prompt("Сколько денег у тебя есть?"))let cena = Number(prompt("Сколько стоит вещь?"))if (deneg >= cena) {  alert("Бери! Сдача останется " + (deneg - cena))} else {  alert("Не хватает " + (cena - deneg) + " — надо подкопить")}

Две дороги через if / else. И там и там программа ещё и считает: сдачу или нехватку.

4Сделай это СВОИМ

  1. Запусти оба калькулятора и проверь на разных числах — ответы меняются?
  2. В калькуляторе денег поставь свою вещь и цену (самокат, игра, подарок)
  3. Поменяй сообщения в alert на свои слова
  4. Добавь третью ветку через else if — если осилишь

✦ Идеи для прокачки

  • Скидка: if (summa >= 1000) alert("Скидка 10%: " + summa * 0.1)
  • Пора спать: спроси час, if (chas >= 22) alert("Спать!") else alert("Ещё можно")
  • Оценка словом: спроси 1–5 и выведи «отлично / хорошо / норм / подтянись»
  • Хватит ли времени: if (est >= nado) alert("Успеешь")

5Покажи и пришли

  1. Покажи родителям — пусть ответят на твои prompt и проверят оба ответа
  2. Сделай скриншот консоли (Print Screen на Windows, Cmd+Shift+4 на Mac)
  3. Пришли скриншот мне в чат — посмотрю, дам совет

✓ Проверь себя

Если калькулятор на одни числа отвечает одно, а на другие — другое, значит if работает и задача выполнена. Красава.

Не работает?

Это нормально, у всех так бывает. Проверь по порядку:

→ Внутри if стоит >= или <, а сравнение чисел — не текста?
→ Каждое число из prompt обернул в Number(...)?
→ После if (...) и else есть фигурные { }?
→ Кавычки английские ", не «ёлочки»?
→ Команды английскими буквами? if, не иф

Если сравнение «врёт» — почти всегда забыт Number(...) вокруг prompt. Совсем застрял — пиши мне.
