Научи программу
решать

Никаких файлов и кнопок. Только браузер, консоль и слово if. Шпаргалка рядом — всё получится.

1Открой консоль

Открой любой сайт (хоть пустую вкладку) и нажми F12 → вкладка Console. На Mac в Chrome — Cmd + Opt + J.

Печатаешь строку → Enter → браузер отвечает. Сегодня он будет ещё и решать.

2Сравни (ответ — true или false)

Набери по очереди и посмотри ответы:

10 > 52 === 2// а это — ловушка, угадай ответ до Enter"100" > "9"

Последняя строка даёт false: текст сравнивается по буквам, не по величине. Вот зачем нужен Number.

3Первое решение: if / else

Заведи коробочку и пусть программа выберет ответ:

let ball = 42if (ball >= 50) {  alert("Зачёт!")} else {  alert("Почти — нужно 50")}

Поменяй 42 на 77 и запусти ещё раз. Ответ изменился?

4Пусть программа спросит и решит

Соедини с прошлым уроком — prompt + Number + if:

let ball = Number(prompt("Сколько баллов набрал?"))if (ball >= 50) {  alert("Зачёт! 🎉")} else {  alert("Почти. Нужно 50, у тебя " + ball)}

Не забудь Number(...) — иначе сравниваешь текст, и программа решит неправильно.

5Сделай это СВОИМ

  1. Придумай, что твоя программа будет решать: зачёт, ранг, «успею/не успею», «хватит денег?»
  2. Поставь свой порог в if (... >= ...)
  3. Напиши свои сообщения в alert для «да» и для «иначе»
  4. Добавь третью ветку через else if — если осилишь

✦ Идеи для прокачки

  • Ранг по очкам: if (xp >= 1000) {...} else if (xp >= 500) {...} else {...}
  • Оценка словом: спроси 1–5, выведи «отлично / хорошо / норм / подтянись»
  • Хватит ли денег: if (deneg >= cena) alert("Бери!")
  • Два условия через &&: «возраст 11–14 — наш Junior»

6Покажи и пришли

  1. Покажи родителям — пусть ответят на твой prompt и проверят оба ответа
  2. Сделай скриншот консоли (Print Screen на Windows, Cmd+Shift+4 на Mac)
  3. Пришли скриншот мне в чат — посмотрю, дам совет

✓ Проверь себя

Если на один ответ программа говорит одно, а на другой — другое, значит if работает и задача выполнена. Красава.

Не работает?

Это нормально, у всех так бывает. Проверь по порядку:

→ Внутри if стоит === (три равно), а не =?
→ Считаешь число из prompt — обернул в Number(...)?
→ После if (...) есть фигурные { }?
→ Кавычки английские ", не «ёлочки»?
→ Команды английскими буквами? if, не иф

Если ничего не выскакивает — почти всегда виноват = вместо ===. Совсем застрял — пиши мне.
