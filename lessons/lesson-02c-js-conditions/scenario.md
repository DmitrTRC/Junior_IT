### 🟡 [0:18 – 0:28] `=` против `===` (кульминация №1)

🎤 «Самое важное и самое коварное за урок.»

```js
let ochki = 5
ochki === 5
ochki === 10
```

🎤 «Один `=` — положить в коробочку. Три `===` — спросить "равно ли?". Перепутаешь внутри условия — программа не спросит, а перезапишет. Это грабля номер один на всю жизнь программиста, честно.»

💬 **В чат:** «🤯 если "= и === — разное" удивило.»

> ⚠️ Не торопись. Тут «щёлкает» понимание всего урока.

---

### 🟡 [0:28 – 0:40] `if` и `else` — две дороги

🎤 «Главное слово: `if` — "если".»

```js
let ball = 80
if (ball >= 50) {
  alert("Зачёт!")
}
```

🎬 Выскочило. Меняешь `ball = 30` — тишина.

🎤 «Условие `false` — браузер пропустил. Добавим "иначе":»

```js
let ball = 30
if (ball >= 50) {
  alert("Зачёт! 🎉")
} else {
  alert("Почти — нужно 50")
}
```

💬 **В чат:** «🔥 если видно, как программа выбирает дорогу.»

> ⚠️ Круглые скобки — условие, фигурные — действие. Покажи обе пары явно.

---

### 🟡 [0:40 – 0:52] Спроси и реши (кульминация №2)

🎤 «Соберём решающую программу целиком.»

```js
let ball = Number(prompt("Сколько баллов набрал?"))
if (ball >= 50) {
  alert("Зачёт! 🎉")
} else {
  alert("Почти. Нужно 50, у тебя " + ball)
}
```

🎬 Введи `72` → зачёт. Перезапусти, `40` → почти.

🎤 «`Number` обязателен: `prompt` отдаёт текст, а текст сравнивается по буквам — `"100" > "9"` соврёт. Спросили → сравнили → решили.»

💬 **В чат:** «Какой свой тест на зачёт придумали бы?»

---

### 🟡 [0:52 – 0:58] Несколько веток (`else if`)

🎤 «Дорог бывает больше двух.»

```js
let xp = Number(prompt("Сколько у тебя XP?"))
if (xp >= 1000) {
  alert("🥇 Золото")
} else if (xp >= 500) {
  alert("🥈 Серебро")
} else {
  alert("🥉 Бронза — качайся дальше")
}
```

🎤 «Браузер идёт по ПЕРВОЙ подошедшей ветке сверху вниз. Это ранги, уровни сложности, оценки — что угодно.»

---

### 🟡 [0:58 – 1:05] Итог + мостик вперёд

🎤 «Что сегодня:»
1. Сравнение даёт да/нет: `>` `<` `===` …
2. `=` положить, `===` сравнить.
3. `if (условие) { ... }` — сделать, если да.
4. `else` — иначе; `else if` — ещё ветка.
5. `prompt` + `Number` + `if` — программа решает сама.

🎤 «На следующем уроке этот `if` повесим на клик кнопки — и получится настоящая реакция и мини-игра.»

## 🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

- **Ничего не выскакивает:** внутри `if` написано `=` вместо `===`? Самая частая причина.
- **Сравнение «врёт»:** забыл `Number(prompt(...))` — текст сравнивается по буквам.

- **`prompt` «завис»:** окошко открылось, но не видно — оно блокирует страницу до ответа.
